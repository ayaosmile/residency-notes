# 执业医师资格考试备考要点

## 结论

执医考试 = **技能考试（6月）+ 综合笔试（8月）**——技能通过后方可参加笔试。技能三道关：病例分析（权重最高）+ 体格检查 + 基本操作。笔试核心：**内+外+妇+儿四科占 ~65%，精神+神经+传染+预防+伦理法规占 ~25%**。

## 技能考试三站

| 考站 | 权重 | 核心备考方向 |
|------|------|------------|
| **第一站：病例分析** | 35% | 按系统归纳（呼吸/循环/消化/泌尿/血液/内分泌/神经/传染/儿科/妇产科约120种疾病）。诊断公式+鉴别诊断+检查+治疗，按模板答。**时间管理是关键**（15min/题） |
| **第二站：体格检查** | 25% | 按系统顺序（一般状况→头颈→胸→腹→脊柱四肢→神经），**口述+手法同时进行**（不口述=0分）。神经检查（12对颅神经+深浅反射+病理征+脑膜刺激征）是扣分重灾区 |
| **第三站：基本操作** | 25% | 四大穿刺+CPR+吸氧+导尿+洗胃+清创+缝合+骨折固定。无菌原则贯穿所有操作（违反1次=全扣）。备物→病人准备→操作流程→术后处理 全程不可跳步 |
| **机考（心肺听诊+影像+心电图+医德医风）** | 15% | 多刷题+多听（心音/呼吸音/肠鸣音）。正常胸片/骨折/气胸/肺炎/胸腔积液 = 必考 |

## 综合笔试

### 分数权重

| 学科 | 占比 | 备考优先级 |
|------|------|----------|
| 内科学 | ~20% | ★★★★★ 最高权重，呼吸/循环/消化各系统均需覆盖 |
| 外科学 | ~15% | ★★★★ 普外+骨科+泌尿+胸外总论 |
| 妇产科学 | ~12% | ★★★★ 正常分娩+病理产科+妇科肿瘤 |
| 儿科学 | ~10% | ★★★★ 生长发育+营养+感染+呼吸+消化 |
| 精神+神经 | ~8% | ★★★ 精神症状学+脑血管病+癫痫 |
| 传染病+预防医学 | ~10% | ★★★ 肝炎+结核+HIV+流感+狂犬+免疫规划 |
| 基础医学（药理/病理/生理/生化/微免） | ~15% | ★★ 药理（重点）+病理（肿瘤/炎症），其余浅覆盖 |
| 医学伦理+卫生法规 | ~5% | ★★ 核心法律+核心伦理原则，可短期突击 |
| 其他（眼科/耳鼻喉/皮肤/影像） | ~5% | ★ 极少单独出题 |

### 备考策略

- **真题为王**：近5年真题至少要完整过2遍（原题重复率~15-20%，考点重复率 > 80%）
- **错题本**：做题不是为了做对，是为了把错题考点整理成册反复看
- **早动手**：从年内开始，每日1-2小时。不比期末考——执医覆盖所有医学领域
- **薄弱科先突击**：大部分考生失分在基础医学+预防+伦理法规
