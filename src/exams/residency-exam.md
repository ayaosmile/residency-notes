# 规培结业考试备考要点

## 结论

规培结业考试 = **理论考试（全国统考）+ 技能考核（OSCE多站式）**——两者均通过方可结业。理论考试专业深度远超执医，覆盖所在专科全部亚专业+相关学科。OSCE = 临床思维+操作能力+沟通人文综合考核。

## 理论考试

### 考试结构

- **题型**：A1（单句型最佳选择题）+ A2（病例摘要型）+ A3/A4（病例组型/病例串型）+ B1（配伍题）+ X型（多选题）——**全部为选择题**
- **题量+时间**：约 150-200 题，2.5-3 小时（依专科不同）
- **出题方向**：所在专科全亚专业覆盖。内科方向 = 呼吸+循环+消化+肾内+血液+内分泌+风湿免疫+感染+神经+急诊+ICU 全覆盖。外科方向 = 普外+骨科+泌尿+胸外+神外+烧伤+麻醉 全覆盖

### 备考策略

- **考纲导向**：规培基地下发的本专科培训细则 = 就是考试范围（所有"掌握"项目都是必考，"熟悉"项目高频考点）
- **指南+共识**：理论考试大量取材于近3-5年的指南——看指南比看教科书更高效
- **刷题平台**：住院医师规范化培训考试题库（人卫版/各地统考系统）
- **年度考核积累**：各基地年度考核+出科考核的题型就是结业考试的缩小版——认真对待每一次

## OSCE技能考核

### 典型考站设计

| 考站 | 时长 | 内容 |
|------|------|------|
| **病史采集** | 8-12 min | SP标准化病人模拟。按系统问诊（主诉→现病史→既往史→个人史→家族史→社会心理）。**不被SP带偏——时间不够时果断转梳理总结** |
| **临床思维** | 15-20 min | 给病例/辅助检查 → 鉴别诊断→诊断→诊疗计划。**不说鉴别诊断 = 不及格** |
| **体格检查** | 10 min | 指定系统（心/肺/腹/神经）。口述+操作并行 |
| **基本操作** | 10-15 min | 穿刺/插管/CPR/骨折固定。无菌原则+适应症+禁忌症+并发症 |
| **沟通人文** | 8-10 min | 坏消息告知/病危通知/医患纠纷沟通情景模拟 |
| **心电图/影像判读** | 10 min | 心梗/心律失常/肺炎/气胸/骨折/脑出血/典型CT |

### 最容易失分的环节
1. **临床思维考站**——只诊断不鉴别、不列出病因、不说明不确定性
2. **无菌原则**——任何操作中违反无菌 = 该站全扣
3. **沟通考站**——太紧张说话快/不共情/不给家属提问和接受信息的时间
4. **时间管理**——在某一步耗时太久导致后续站崩
5. **过度紧张**——OSCE是表演，不等同于真实临床。SP是给信息的人不是敌人
