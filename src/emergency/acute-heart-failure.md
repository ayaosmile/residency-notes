# 急性心力衰竭的识别与处理

## 结论

突发呼吸困难 + **双肺湿啰音 + NT-proBNP > 300 pg/mL（排除值）/ 年龄分层诊断值** → 急性心衰 → **坐位+给氧+利尿+扩血管四步同时启动** → 低氧/呼吸衰竭 → 无创通气。

## 核心要点

- **NT-proBNP解读**（ESC 2021/2023 HFA共识）：
  - < 300 pg/mL → **排除急性心衰**（NPV > 98%）
  - < 50 岁：诊断值 > 450 pg/mL；50-75 岁：> 900；> 75 岁：> 1800 pg/mL
  - 肥胖患者（BMI > 35）→ 诊断阈值需下调约 50%
- **30秒床旁评估**：坐位（不能平卧 = 端坐呼吸）+ SaO₂ + 颈静脉怒张 + 双肺对称性湿啰音（心源性 vs 单侧/不对称 = 感染/其他）+ 下肢水肿 + 皮肤湿冷
- **处理四步同启**：
  1. **坐位垂腿**——减少回心血量
  2. **给氧**——SaO₂ < 90% 或 PaO₂ < 60 → 无创正压通气（CPAP/BiPAP）；Ⅱ型呼衰 → BiPAP
  3. **利尿**——呋塞米 20-40 mg IV（未使用过利尿剂）或原有口服剂量 2.5 倍 IV（慢性心衰加重）；无反应 → 加倍，最大 160-200 mg IV
  4. **扩血管**——硝酸甘油 10-20 μg/min 起始静脉输注（如 SBP > 110）或 硝酸异山梨酯 1-2 mg/h IV
- **低血压（SBP < 90）**：不用扩血管药 → 正性肌力药（多巴酚丁胺 2.5-5 μg/kg/min 或 米力农）
- **警惕触发器**：AMI、快速房颤、高血压急症、感染、肺栓塞、容量超负荷（饮食/药物依从性）、贫血、甲亢
