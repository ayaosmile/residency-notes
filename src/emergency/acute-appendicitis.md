# 急性阑尾炎的诊断与处理

## 结论

转移性右下腹痛 + **右下腹压痛反跳痛** → 急性阑尾炎（临床诊断）→ **Alvarado评分/SRS评分分层** → 高分/中分+CT确诊 → 急诊腹腔镜阑尾切除术。非穿孔性且无粪石的低风险成人可考虑抗生素保守治疗（知情选择下）。

## 核心要点

### 诊断

- **Alvarado评分**（10分制）：迁移痛(1)+食欲减退(1)+恶心呕吐(1)+右下腹压痛(2)+反跳痛(1)+发热>37.3(1)+WBC>10×10⁹/L(2)+中性粒>75%(1)
  - ≥ 7：高危——手术；**5-6**：中危——CT；**≤4**：低危——观察/出院
- **CT（金标准）**：阑尾直径 > 6 mm、壁增厚/强化、阑尾周围脂肪条索、粪石。敏感性 98%，特异性 97%
- **超声（育龄女性/儿童首选）**：无辐射。敏感性 85%，特异性 90%
- **妊娠**：首选超声 → 不确定则MRI。二孕期手术风险最低，抗生素可尝试（成功率 ~80%）

### 治疗

| 类型 | 方案 |
|------|------|
| **单纯性阑尾炎（非穿孔）** | ①**标准=腹腔镜阑尾切除术**（LA）——24h内 ②非手术——抗生素保守治疗（适用于无粪石+阑尾直径≤11mm+知情选择）。厄他培南1g q24h或头孢曲松2g+甲硝唑500mg q8h IV×7-10d。30天成功率~70%，1年~60% |
| **穿孔性阑尾炎** | 急诊手术+充分腹腔灌洗。术后抗生素：哌拉西林他唑巴坦4.5g q6h或头孢曲松+甲硝唑×3-5d |
| **阑尾周围脓肿** | **非手术=经皮引流+抗生素**（急诊手术造瘘风险高）。静脉抗生素+介入引流→6-12周后择期阑尾切除或随访观察 |

### 围术期抗生素

- **单纯性**：术前单次头孢唑林2g IV+甲硝唑500mg IV → 术后 **无需继续抗生素**（Surgical care improvement）
- **穿孔/坏疽**：术后抗生素 3-5d（哌拉西林他唑巴坦/头孢曲松+甲硝唑）
