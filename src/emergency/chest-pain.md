# 急性胸痛的快速鉴别

## 结论

急性胸痛 → **第一时间做 ECG + 肌钙蛋白** → 排除 STEMI（ST 抬高 → 再灌注）→ 按致命性病因依次排除：主动脉夹层、肺栓塞、张力性气胸、心包填塞。

## 核心要点

- **0 分钟 — ECG**：ST 抬高 ≥ 2 个相邻导联 → STEMI → 立即再灌注（PCI < 90 min 或溶栓）
- **0 分钟 — 生命体征**：双上肢血压差 > 20 mmHg → 警惕主动脉夹层；休克 + 颈静脉怒张 → 心包填塞；单侧呼吸音消失 + 气管偏移 → 张力性气胸
- **30 分钟 — 肌钙蛋白 hs-cTn**：阴性不排除 ACS，需 1-3h 复查
- **床旁超声（eFAST）**：右室扩大 + 室间隔矛盾运动 → 肺栓塞；心包积液 → 填塞；主动脉根部扩张 → 夹层
- **胸痛 5 大致命病因**（每 15 min 复查一次）：AMI、主动脉夹层、肺栓塞、张力性气胸、心包填塞
