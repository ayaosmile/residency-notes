# 气胸的识别与紧急处理

## 结论

突发单侧胸痛+呼吸困难 + **患侧呼吸音消失 + 气管偏向对侧** → 张力性气胸（临床诊断，不等胸片）→ **立即第2肋间锁骨中线 粗针穿刺减压（14-16G）** → 胸腔闭式引流。

## 核心要点

- **张力性气胸 = 临床诊断，不是放射学诊断！** 三联征：呼吸窘迫 + 患侧呼吸音消失 + 气管偏移 → 不等胸片，不等CT → 立即针减压
- **张力性气胸 — 针减压**：**第2肋间锁骨中线（安全三角区上缘）** 或 **第5肋间腋前线**（锁骨中线外侧 2 cm），14-16G 留置针垂直进针 → 听到气体逸出声 → 留套管开放 → 立即准备胸腔闭式引流
- **非张力性气胸 — 处理按大小**：
  - **少量（< 2 cm）且无症状**：观察+高流量吸氧（加快空气吸收 3-4倍），24-48h 复查胸片
  - **大量（≥ 2 cm）或有症状**：胸腔闭式引流（第5肋间腋前线/腋中线，18-24F引流管）
  - **或**：小口径穿刺抽吸（8-14F导管，首次抽吸成功率高），复发则换引流管
- **继发性气胸（COPD/ILD等基础肺病）**：即使少量也应引流——肺储备差，耐受性低
- **医源性气胸**（中心静脉穿刺后、机械通气中）→ 引流 + 机械通气参数调整（降低潮气量+PEEP）
- **出院指征**：引流无气泡 > 24h + 胸片肺复张 → 夹管4-6h 复查胸片 → 持续复张 → 拔管
