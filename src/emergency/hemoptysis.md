# 大咯血的识别与紧急处理

## 结论

24h 咯血量 > 200-600 mL 或 每小时 > 100 mL → **患侧卧位（保护健侧肺）** → 气道保护（GCS ≤ 8 插管，必要时双腔管健侧肺隔离）→ **支气管动脉栓塞（BAE，首选）** 或 支气管镜下止血 → 手术（BAE失败/复发时）。

## 核心要点

- **定义**：大咯血 = 24h 咯血量 > 200-600 mL 或有任何危及生命的气道/氧合威胁（出血量 < 200 mL 但伴呼吸衰竭也按大咯血处理）
- **鉴别**：咯血（咳嗽带血，泡沫状，pH碱性）vs 呕血（呕吐带血，咖啡渣样，pH酸性，无气泡）
- **第一动作**：**患侧卧位**（已知出血侧→朝下，重力引流，保护健侧肺；未知出血侧→仰卧头低位）
- **气道管理**：
  - 清醒+有力咳嗽 → 自主排血
  - 意识下降/无力排血/GCS ≤ 8 → 气管插管（使用 8.0 mm 以上大管径便于吸痰/支气管镜）
  - 持续健侧污染 → 双腔管 → 单侧肺隔离
- **首选出止血方案：支气管动脉栓塞（BAE）**
  - 技术成功率 > 90%，临床止血率 70-99%
  - 不良反应率低（< 1% 截瘫风险 — 脊髓动脉栓塞）
- **支气管镜**：可定位出血部位 + 球囊填塞/冰盐水灌洗/局部止血药物 → 作为 BAE 前过渡或术中定位
- **药物辅助**：氨甲环酸 500-1000 mg IV（抗纤溶）；垂体后叶素 5-10 U + 5% GS 20 mL IV 缓慢（收缩血管，高血压注意）
- **手术（肺叶切除）**：BAE 2 次失败、持续大咯血、或存在可切除局限病灶 → 择期手术优于急诊（急诊死亡率 16-46%）
