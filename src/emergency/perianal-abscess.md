# 肛周脓肿的诊断与处理

## 结论

肛门周围剧痛 + **局部红肿热痛+波动感/硬结** → 肛周脓肿（临床诊断——通常不需要影像！）→ **切排引流（唯一有效治疗）** → 切口要够大（防早期闭合）→ 术后坐浴+抗生素仅限特殊人群。

## 核心要点

### 分型（按解剖间隙）

| 类型 | 位置 | 特点 |
|------|------|------|
| **肛周皮下脓肿** | 肛门边缘（最常见~60%） | 表浅、波动感明显、疼痛剧烈（但局限） |
| **坐骨直肠窝脓肿** | 距肛缘3-5cm，深部 | 范围大、高热+全身中毒表现明显、**表面可能无波动感** |
| **括约肌间脓肿** | 内外括约肌间隙 | 指诊可触及直肠壁内痛性隆起。常被漏诊（外观几乎正常） |
| **提肛肌上脓肿** | 骨盆直肠间隙（最危险！） | 极少见。可延伸至腹腔→盆腔腹膜炎。发热+全身中毒但**局部可见体征极轻微**——CT确诊 |

### 处理原则

- **唯一有效治疗 = 切排引流！** 抗生素不能替代引流（脓肿=封闭空腔=抗生素无法进入）
- **切开引流要点**：
  - 切口位置：尽量靠近肛缘（以缩短未来可能形成的瘘管）
  - 切口长度：**脓肿腔直径的1/2-2/3**——足够大、防止过早闭合（常见失败原因=切口太小→脓腔未完全打开/引流不畅）
  - 钝性分离脓腔分隔（不要锐性盲目剥离——避免损伤括约肌）
  - **不要填塞过紧**——填塞物取出后疼痛剧烈+不真正改善引流
  - 留取脓液送培养（尤其复发/复杂脓肿——排查MRSA/真菌）
- **术后管理**：
  - 温水坐浴qid+便后 → 持续引流至创面愈合（一般3-6周由内向外）
  - 镇痛（NSAIDs通常足够）+ 大便软化剂（乳果糖15-30mL qd-bid）
  - 1-2周门诊随访 → 评估创面愈合+排查肛瘘

### 抗生素指征（不是常规！）

| 需用抗生素的人群 | 方案 |
|---------------|------|
| **广泛蜂窝织炎**（脓肿周围大片红肿热痛） | 阿莫西林克拉维酸875mg bid PO或头孢氨苄500mg qid PO |
| **免疫抑制/糖尿病** | 切排后口服抗生素7d预防全身感染 |
| **心脏瓣膜病/人工瓣膜** | 抗生素预防心内膜炎（阿莫西林2g PO术前1h） |
| **全身中毒/发热** | 静脉抗生素（头孢曲松2g q24h+甲硝唑500mg q8h） |
| **MRSA风险** | 克林霉素300-450mg tid PO或SMZ-TMP DS bid PO |

- **单纯肛周脓肿切排后不需要抗生素**——多项RCT证实无额外获益

### 肛瘘

- **肛周脓肿的必然伴侣**：约30-50%切排后会形成肛瘘（肛门隐窝→脓肿→外口管道持续不愈）
- **肛瘘诊断线索**：术后>6周仍有间歇性渗液/分泌物从原切口流出 → 肛瘘
- **处理**：肛瘘切除术/挂线疗法（如累及括约肌——挂线保护肛门功能）
