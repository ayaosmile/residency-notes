# 急性腹痛的快速评估

## 结论

急性腹痛 → **生命体征 + 腹部查体（腹膜刺激征=外科！）** → 按**部位分层**鉴别（左上/右上/中上/脐周/左下/右下/弥漫）→ 高危人群（老年/免疫抑制/妊娠）即使体征轻微也不能掉以轻心。

## 核心要点

- **外科腹痛红牌（立即外科会诊）**：板状腹/反跳痛/肌紧张（腹膜刺激三联征）+ 血流动力学不稳定 + 消化道穿孔游离气体（立位腹平片/CT）
- **按部位鉴别**：

| 部位 | 常见病因 |
|------|---------|
| 右上腹 | 胆囊炎/胆石症、胆管炎、肝脓肿、膈下脓肿 |
| 中上腹 | 胰腺炎、消化性溃疡穿孔、胃炎、心梗（下壁！） |
| 右下腹 | 阑尾炎、Meckel憩室、卵巢扭转、异位妊娠 |
| 弥漫 | 肠梗阻、穿孔、肠系膜缺血、腹主动脉瘤破裂 |

- **关键4问**：①腹痛位置+有无放射（背部=胰腺/AAA，肩=膈下）②有无发热（感染 vs 非感染）③有无停止排气排便（肠梗阻）④末次月经（育龄女性 = 排除异位妊娠！）
- **诊断性检查分层**：
  - **基础**：血常规+CRP、肝肾功能+淀粉酶/脂肪酶、尿常规+HCG（育龄女性）、立位腹平片（游离气体）
  - **进阶**：腹部CT平扫+增强（金标准）；超声（胆囊炎/阑尾炎/妇科）；CTA（肠系膜缺血/AAA）
- **镇痛不影响诊断**：合理剂量的阿片类镇痛药不增加误诊率。让患者在剧痛中等待检查是不合理的
