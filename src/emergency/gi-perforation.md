# 消化道穿孔的识别与处理

## 结论

突发剧烈腹痛 + **腹膜刺激征（板状腹）** → **立位胸片/腹部平片示膈下游离气体**（灵敏度~70%，CT灵敏度~95%）→ 消化道穿孔 → **禁食+输液+广谱抗生素+急诊外科手术**。

## 核心要点

### 诊断

- **症状三联征**：突发电击样腹痛（穿孔时刻明确）+ 弥漫性腹膜炎（板状腹/全腹反跳痛）+ 发热/休克（随病情进展）
- **立位胸片**：双侧膈下新月形透亮带=游离气体=穿孔=外科立即会诊。注意：**无游离气体不能排除穿孔**（30%假阴性，尤其是十二指肠球后/结肠后壁穿孔或小穿孔）
- **CT腹部平扫+增强**：游离气体+液体积聚+穿孔部位壁增厚/缺损。灵敏度~95%。被包裹的穿孔（壁内气体）也可检出
- **实验室**：WBC↑/核左移、乳酸↑（组织低灌注/感染性休克）、Cr↑（AKI）

### 紧急处理

| 步骤 | 操作 |
|------|------|
| **1. 稳定循环** | **LR/NS 1-2L bolus+ MAP目标>65**。脓毒性休克？→启动SSC 1h Bundle |
| **2. 禁食+NGT** | 持续胃肠减压（减少胃内容物继续漏入腹腔） |
| **3. 抗生素** | **立即启动**（30min-1h内）：哌拉西林他唑巴坦4.5g q6h或头孢曲松2g q24h+甲硝唑500mg q8h。覆盖G⁻杆菌+厌氧菌。重症→美罗培南1g q8h |
| **4. 外科会诊** | **立即**（不要等抗生素起效再叫！）。穿孔位置+原因+患者状态决定手术方式 |
| **5. 镇痛** | 芬太尼25-50μg IV或吗啡2-5mg IV。镇痛不碍诊断、不增加死亡率 |

### 手术决策

| 穿孔类型 | 手术方式 |
|---------|---------|
| **消化性溃疡穿孔（胃/十二指肠）** | 腹腔镜下带血管蒂大网膜填塞修补+腹腔灌洗（Graham patch）——标准术式。术后PPI+HP检测+根除 |
| **结肠穿孔** | ①切除+一期吻合（选择性）+保护性造口 或 ②Hartmann手术（切除+末端结肠造口）——穿孔合并粪便污染者首选 |
| **阑尾穿孔** | 腹腔镜阑尾切除+腹腔灌洗。见[急性阑尾炎](./acute-appendicitis) |
| **憩室穿孔** | ①Hinchey Ⅰ/Ⅱ=切除+一期吻合±腹腔镜灌洗 ②Hinchey Ⅲ/Ⅳ=Hartmann术式或切除+吻合+保护性造口 |

### 关键提示

- **上消化道穿孔的"蜜月期"**：穿孔后数小时腹痛可暂时缓解（胃酸被漏出的液体稀释）→ 6-12h后细菌性腹膜炎→疼痛复发加重。不要被蜜月期骗了！
- **老年/免疫抑制/激素使用者**：体征可能极轻微（无板状腹）——这类患者的任何腹痛+休克/发热都应立即CT排查
- **保守治疗可能适用（极少量选择性病例）**：小穿孔<5mm+造影显示已闭合+无全身感染+无腹膜炎——胃十二指肠小穿孔可禁食+PPI+抗生素保守管理
