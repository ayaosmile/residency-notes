# 高血压急症的评估与处理

## 结论

BP ≥ 180/120 mmHg + **靶器官损害证据**（脑/心/肾/眼底/大血管）→ 高血压急症 → 收ICU → **缓慢降压**（第1h降 ≤ 25%，2-6h 至 160/100-110，24-48h 至正常）。无靶器官损害 = 高血压亚急症 → 口服降压 + 门诊随访，**无需紧急静脉降压**。

## 核心要点

- **区分急症 vs 亚急症**：关键不是血压数值，而是 **有无靶器官损害**。血压 220/140 无症状 ≠ 急症；血压 180/110 + 意识模糊 = 急症
- **靶器官损害快速筛查**：眼底（视乳头水肿/出血）、头颅CT（脑出血/脑水肿）、ECG+肌钙蛋白（心肌缺血）、Cr（AKI）、胸片（肺水肿/主动脉增宽）
- **降压目标**：**第1小时降 ≤ 25%**（过快→脑/心/肾低灌注）→ 2-6h 至 160/100-110 → 24-48h 平稳至正常
- **首选药物**：
  - 主动脉夹层 → 艾司洛尔/拉贝洛尔 + 硝普钠（目标 HR < 60，SBP < 120 或最低可耐受）
  - 脑出血 → 尼卡地平/乌拉地尔（SBP > 220 时降至 < 180；SBP 150-220 时降至 140-160）
  - 急性心衰 → 硝酸甘油 + 呋塞米
  - 其他→尼卡地平/乌拉地尔/拉贝洛尔
- **禁忌**：**禁用短效硝苯地平口服/舌下含服**（不可控骤降→卒中/MI）
- **亚急症处理**：口服降压药（CCB/ACEI/ARB起始），调整现有方案，门诊 1 周内随访
