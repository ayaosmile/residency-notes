# 腹股沟疝嵌顿的识别与处理

## 结论

腹股沟区**不可回纳的包块 + 疼痛** → 嵌顿疝 → **评估绞窄征象（剧痛/皮肤变色/发热/肠梗阻）** → 无绞窄 → 尝试手法复位（镇静+Trendelenburg位）→ 复位成功→限期疝修补；复位失败或绞窄 → **急诊手术（<4-6h）**。

## 核心要点

### 诊断与绞窄判断

- **嵌顿 = 不可回纳 + 无肠缺血**；**绞窄 = 嵌顿 + 血流中断 → 肠坏死**
- **绞窄征象**（有任一条 = 禁止手法复位 → 直接手术）：
  - 包块触痛剧烈（疼痛与疝大小不成比例）+ 皮肤红肿/发热/青紫变色
  - **肠梗阻征象**（恶心呕吐/腹胀+停止排气排便）——小肠嵌顿→梗阻
  - 发热+WBC↑+乳酸↑+代酸 → 肠坏死
  - 超声示肠壁血流消失（多普勒阴性）——但超声不能替代临床判断

### 手法复位

- **仅适用于**：无绞窄征象 + 嵌顿时间<4-6h + 患者镇静配合
- **方法**：
  1. 镇静镇痛：吗啡2-5mg IV或芬太尼25-50μg IV+咪达唑仑1-2mg→**腹壁完全放松**
  2. **Trendelenburg位**（头低脚高15-20°）→利用重力辅助回纳
  3. 一手固定疝环口→另一手轻柔持续施压（从疝囊远端向疝环口推挤，非暴力！）
  4. 复位成功=听见"咕噜"声+包块消失+腹痛立即缓解→观察2-4h+收住院
- **复位失败/绞窄** → **急诊手术**（<4-6h）。延迟=每一小时肠坏死风险增加
- **复位成功后的处理**：择期疝修补术（腹腔镜TEP/TAPP或开放Lichtenstein）→同次住院或尽早（<2周）。复位后出现腹膜刺激征→立即手术

### 手术方式

| 场景 | 术式 |
|------|------|
| **绞窄（肠坏死）** | 急诊剖腹/腹腔镜 → 切除坏死肠段±吻合 ± 造口（污染/休克时Hartmann式）。疝修补用自体组织（感染环境下不用网片！——补片感染率极高） |
| **嵌顿（肠存活）** | 复位+疝修补。组织条件好→网片修补（Lichtenstein），组织水肿/污染→Bassini/Shouldice自体组织修补 |
| **股疝嵌顿** | 尤其危险（疝环紧、坏死率↑、容易漏诊）→ **任何嵌顿股疝=立即手术**（不尝试手法复位——复位成功率低+坏死率高） |

### 关键提示

- **股疝**：位于腹股沟韧带下方+股三角区，小而紧——即使极小包块也可含坏死肠段。漏诊率最高（误诊为淋巴结炎/股疝不认识）→CT可避免漏诊
- **复位前超声**：非必须但如超声显示肠蠕动+血流→可尝试复位。血流缺失→手术
- **手法复位后必须收住院观察**：即使复位成功也需观察肠恢复（进食+排气+腹痛缓解后出院）
