# 急性心律失常的处理

## 结论

心律失常 → **第一条问题：患者血流动力学是否稳定？** → 不稳定 = **同步电复律/起搏（不要等药！）** → 稳定 = 12导联ECG分类（窄QRS vs 宽QRS，规整 vs 不规整）→ 病因导向药物。

## 核心要点

### 决策第一问：稳定 vs 不稳定？

**不稳定征象**（有任一项 = 不稳定 → 电复律/起搏）：
- 休克（SBP < 90，组织低灌注）
- 意识障碍（晕厥/濒临意识丧失）
- 急性心衰（肺水肿）
- 心肌缺血（胸痛+ST改变）
- 极快心率 > 150 bpm（心输出量下降）

### 心动过速

| 类型 | 稳定处理 | 不稳定处理 |
|------|---------|-----------|
| **窄QRS规整** | 迷走神经刺激 → 腺苷 6 mg IV 快速推注（可 12 mg → 18 mg） | 同步电复律 50-100J |
| **窄QRS不规整（房颤）** | 控制心率（美托洛尔 2.5-5 mg IV / 地尔硫卓 0.25 mg/kg IV）；< 48h 可考虑药物复律（胺碘酮/伊布利特）或电复律；≥ 48h → 除外左房血栓再复律（TOE/抗凝3周优先） | 同步电复律 120-200J |
| **宽QRS规整（VT？SVT+BBB？）** | 默认按 VT 处理（安全原则）→ 胺碘酮 150 mg IV 10 min → 1 mg/min × 6h → 0.5 mg/min × 18h | 同步电复律 100J（单相） |
| **多形性VT/Torsades de Pointes** | **MgSO₄ 2g IV push** + 纠低钾 + 超速起搏 | **非同步电除颤**（按室颤处理：200J 双相） |

### 心动过缓

- **不稳定**：**阿托品 0.5 mg IV（可重复至 3 mg）** → 无效 → 经皮起搏/经静脉起搏
- **稳定+高危**：二度Ⅱ型/三度AVB + 宽QRS逸搏 → 即使暂时稳定也准备起搏
- **药物相关**：β阻滞剂/CCB/地高辛过量 → 胰高血糖素/钙剂/地高辛抗体
- **起搏等待期**：多巴胺 5-20 μg/kg/min 或 肾上腺素 2-10 μg/min 可临时维持

### 关键提示

- **不稳定 = 电复律/起搏，不等药！** 这是ERC/ACLS最强调的原则
- **宽QRS = VT 直到证明不是**：宁可把SVT+BBB按VT误治，也不可把VT按SVT误治（维拉帕米→VT→崩溃）
- **房颤 < 48h**：可安全复律（药物/电）；**≥ 48h 或时间不明确** → 抗凝 3 周再复律 或 TOE 排除血栓后即刻复律
