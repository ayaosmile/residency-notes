# 意识障碍的床旁评估

## 结论

意识障碍 → **ABC + GCS评分 + 快速血糖**（低血糖是首要可逆病因）→ 按昏迷病因"**4H-4T**"排查 → 保护气道（GCS ≤ 8 插管）→ 病因导向治疗。

## 核心要点

- **GCS（Glasgow Coma Scale）**：睁眼（1-4）+ 语言（1-5）+ 运动（1-6）= 3-15 分

| GCS | 意义 | 处理 |
|-----|------|------|
| 13-15 | 轻度 | 观察，查找病因 |
| 9-12 | 中度 | 密切监护 |
| ≤ 8 | **重度/昏迷** | **保护气道→气管插管（≤8=插管！）** |

- **昏迷病因 — "4H-4T" 排查法**：
  - **4H**：Hypoxia（缺氧，查SaO₂+ABG）、Hypoglycemia（低血糖，**指尖血糖1分钟即得！**）、Hypovolemia/Hypotension（低血压/休克）、Hyper/Hypothermia（高热/低体温）
  - **4T**：Trauma（外伤，查头部面部伤痕）、Toxins（中毒/药物过量，问服药史）、Thrombosis（卒中/SAH，急诊CT平扫）、Tumor/Infection（占位/CNS感染，CT+腰穿）
- **生命体征线索**：
  - 高血压+心动过缓+Cushing呼吸 → ICP升高（Cushing三联征）→ 急诊CT
  - 高热+颈强直 → 脑膜炎/脑炎 → 血培养+抗生素+腰穿
  - 低体温 → 甲减/低血糖/酒精/脓毒症
- **格拉斯哥评分之外**：瞳孔（大小+对光反射 = 脑干功能）、呼吸模式（Cheyne-Stokes → 半球病变；Kussmaul → 酸中毒；不规律 → 延髓损伤）
- **不能忘记的一步**：**查药物/毒物史、酒精、基础病史**（糖尿病/肝病/癫痫/精神疾病/抗凝）
