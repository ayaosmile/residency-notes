# 急性发热的评估思路

## 结论

体温 > 38.3°C（或 > 38°C 持续 > 1h） → **qSOFA/SIRS 快速脓毒症筛查** → 血培养（抗生素前！2 套，不同部位）+ 感染源导向检查 → 脓毒症/脓毒性休克 → **1h内启动抗生素（Bundle 1h）**。

## 核心要点

- **qSOFA 筛查（床旁30秒）**：呼吸 ≥ 22 + SBP ≤ 100 + GCS < 15 → ≥ 2 分 = 高危 → 评估脓毒症
- **脓毒症诊断**：感染 + SOFA 急性变化 ≥ 2 分（或 qSOFA ≥ 2 + 感染证据）
- **脓毒性休克**：脓毒症 + 充分补液后仍需升压药维持 MAP ≥ 65 + 乳酸 > 2 mmol/L → 死亡率 > 40%
- **"1小时Bundle"（Surviving Sepsis Campaign 2024）**：
  1. 测血乳酸 → 1h 内复测（首次升高）
  2. **抗生素前**取血培养（2 套，不同部位，需氧+厌氧）
  3. 启动广谱抗生素（1h内！每延迟1h死亡率 + 7-9%）
  4. 低血压/乳酸 ≥ 4 → 晶体液 30 mL/kg 快速复苏
  5. 充分补液后仍低血压 → 去甲肾上腺素 0.05-0.1 μg/kg/min
- **发热待查（FUO）经典定义**：体温 > 38.3°C × > 3 周 + 住院 1 周无明确诊断 → 5 大类：感染（30-40%）、肿瘤（20-25%）、结缔组织病（15-20%）、其他（10-15%）、未确诊（10-30%）
- **特殊人群**：粒缺伴发热（ANC < 0.5×10⁹/L + 单次体温 ≥ 38.3°C 或 ≥ 38°C 持续 > 1h）→ **不等待血常规/CRP结果，立即广谱抗生素**
