# 低血糖的识别与处理

## 结论

血糖 < 3.9 mmol/L → 评估意识状态 → **意识清楚**：口服 15-20g 速效碳水 → 15 min 复测；**意识障碍**：50% GS 40-60 mL IV → 持续输注 10% GS 维持。

## 核心要点

- **诊断标准（中国2型糖尿病防治指南）**：血糖 < 3.9 mmol/L（低血糖）；血糖 < 3.0 mmol/L（严重低血糖，需他人协助）
- **Whipple三联征**：低血糖症状 + 实测血糖低 + 补糖后症状缓解
- **意识清楚患者**："**15-15 法则**"
  - 15 g 速效碳水（4块方糖/150 mL 果汁/3-4片葡萄糖片）
  - 15 min 后复测血糖 → 未达标重复 → 达标则进食慢速碳水（饼干/面包）防止复发
- **意识障碍患者（急症！）**：
  - 50% GS 40-60 mL IV 推注 → 复测血糖
  - 如意识未恢复 → 10% GS 持续输注（100 mL/h），维持血糖 5-10 mmol/L
  - **可同时肌注胰高糖素 1 mg**（尤其静脉通路困难时，10-15 min 起效）
- **特殊人群**：
  - 使用长效磺脲类（格列本脲等）→ 低血糖可持续 24-72h → 收住院 + 持续糖输注
  - 酒精相关性低血糖 → 补充硫胺素（维生素 B1 100 mg IV）预防 Wernicke 脑病
- **病因探查**（反复低血糖）：药物（胰岛素/磺脲类最常见）、酒精、肝衰竭、胰岛素瘤、肾上腺皮质功能不全
