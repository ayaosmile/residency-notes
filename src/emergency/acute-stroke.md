# 急性脑卒中的急诊评估与处理

## 结论

突发神经功能缺损 → **NIHSS评分 + 急诊头颅CT平扫（首诊15-25min内完成）** → 排除出血 → 缺血性卒中 → **发病 < 4.5h 且无禁忌 → 阿替普酶（rt-PA）0.9 mg/kg IV**（10% bolus + 90% 输注 60 min）。

## 核心要点

- **院前FAST筛查**：Face（面瘫）→ Arms（单侧无力）→ Speech（言语含糊）→ Time（记录发病时间，越快越好）
- **急诊CT的第一任务：排除脑出血（不是确诊脑梗死！）** 早期缺血CT可完全正常。溶栓决策基于临床+排除出血，不等MRI
- **溶栓窗**：
  - **标准窗 ≤ 4.5h**：rt-PA 0.9 mg/kg（最大 90 mg），10% 1 min iv bolus + 90% 60 min 输注
  - **醒后卒中/时间不明（4.5-9h）**：MRI DWI-FLAIR mismatch（DWI阳性+FLAIR阴性）→ 可考虑溶栓（WAKE-UP标准）
  - **大血管闭塞 6-24h**：机械取栓（CTP/RAPID筛选）
- **溶栓前血压控制**：BP < 185/110 mmHg 方可溶栓（硝酸甘油/尼卡地平降压 → 达标后溶栓；溶栓后维持 < 180/105 × 24h）
- **出血性卒中**：
  - 停用抗凝/抗血小板 → 凝血功能急查
  - 华法林相关 → Vit K 10 mg IV + PCC（4因子凝血酶原复合物）
  - SBP > 220 → 尼卡地平/乌拉地尔降压（第1h < 180）；SBP 150-220 → 缓降至 140-160
  - 脑叶血肿 > 30 mL + GCS 下降 → 神经外科手术评估
- **NIHSS关键阈值**：0-4 轻度；5-15 中度；16-20 中重度；≥ 21 重度
