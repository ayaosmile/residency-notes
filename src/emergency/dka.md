# 糖尿病酮症酸中毒的识别与处理

## 结论

高血糖 + **阴离子间隙增高性代谢性酸中毒 + 酮体阳性** → DKA → **0.9% NS 快速补液（第1h 15-20 mL/kg）** → 胰岛素持续输注（0.1 U/kg/h，血糖降至 13.9 mmol/L 后改用 GS+胰岛素维持）→ **补钾（几乎必需！）** 。

## 核心要点

- **诊断标准**：血糖 > 13.9 mmol/L（通常 16.7-33.3）+ pH < 7.30 / HCO₃⁻ < 18 mmol/L + 血酮 ≥ 3.0 mmol/L 或尿酮 ≥ ++ → 三项全部满足 = DKA
- **严重程度**：pH 7.25-7.30 = 轻度；7.00-7.24 = 中度；< 7.00 = 重度
- **补液（最优先！）**：0.9% NS 第 1h 15-20 mL/kg → 之后 250-500 mL/h（根据心肾功能调整）。24h 总液量 4-6 L。血糖降至 13.9 mmol/L 时换 5% GS + 0.45% NS（防止低血糖+脑水肿）
- **胰岛素**：**先确认血钾 ≥ 3.3 mmol/L 再启动！** 短效胰岛素 0.1 U/kg IV bolus → 0.1 U/kg/h 持续输注。血糖下降速度 3-4 mmol/L/h；如第 1h 未降 10%，翻倍胰岛素剂量
- **补钾（几乎每个DKA都要补！）**：起始胰岛素治疗时血钾因酸中毒正常或偏高，但**全身钾储备已严重缺失**。K⁺ < 5.3 mmol/L → 开始补钾 20-40 mmol/L 加入液体；K⁺ < 3.3 → 胰岛素延后，先补钾
- **HCO₃⁻ 仅限 pH < 6.9 时考虑**（50-100 mmol 缓慢输注），常规使用增加脑水肿风险
- **脑水肿预警**（儿童高发，成人也可发生）：**头痛、意识恶化、心率变慢** → 立即甘露醇，暂停胰岛素，减慢输液
