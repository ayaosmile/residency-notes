# 急性呼吸困难的床旁评估

## 结论

急性呼吸困难 → **快速判断类型**（低氧性 vs 高碳酸性 vs 混合性）→ **床旁 SpO₂ + ABG + 肺部听诊 + 颈静脉** → 排查 6 大致命病因：心衰、COPD/AE、哮喘、肺栓塞、气胸、气道梗阻。

## 核心要点

- **看+听 30 秒**：呼吸频率（> 30 或 < 8 → 即将衰竭）、辅助呼吸肌参与、说话能力（能说完整句子 → 尚可；单字 → 即将插管）
- **SpO₂**：< 90% → 立即给氧（鼻导管 2-4L → Venturi → HFNC）
- **ABG/VBG**：PaO₂ < 60 mmHg（I型呼衰）；PaCO₂ > 50 mmHg + pH < 7.35（II型呼衰）；A-a gradient 用于区分肺实质 vs 通气不足
- **肺听诊 4 象限**：弥漫湿啰音 → 心衰；哮鸣音 → COPD/AE 或哮喘；单侧消失 → 气胸/大量胸腔积液
- **颈静脉**：怒张 → 心衰/肺栓塞/心包填塞；塌陷 → 低血容量
- **下肢**：不对称肿胀 → 警惕 DVT/肺栓塞
