# 急性低血压的评估与处理

## 结论

收缩压 < 90 mmHg 或 MAP < 65 mmHg 持续 > 5 分钟 → **立即评估组织灌注**（意识、尿量、皮肤、乳酸）→ 按休克分型处理。

## 核心要点

- **第一步：确认血压真实**——复测对侧上肢 + 手动血压计核实
- **第二步：快速灌注评估**——意识（GCS）、尿量（< 0.5 mL/kg/h 为少尿）、皮肤（花斑、湿冷）、乳酸（> 2 mmol/L 提示灌注不足）
- **第三步：分型处理**
  - 低血容量性 → 补液试验（晶体液 500 mL iv 15 min）+ 控制活动性出血
  - 心源性 → 强心/血管活性药 + 查找心梗/心衰病因
  - 分布性（感染性）→ 去甲肾上腺素 + 抗感染 + 液体复苏
  - 梗阻性 → 解除梗阻（心包穿刺/溶栓/胸腔减压）
- **补液试验**：晶体液 500 mL 静脉 15 min，观察血压反应 + 肺部啰音
- **去甲肾上腺素**：起始 0.05-0.1 μg/kg/min，经中心静脉给药；无中心静脉时可短期经粗大外周静脉
