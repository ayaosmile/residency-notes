# 肠梗阻的诊断与处理

## 结论

腹痛+腹胀+**停止排气排便+CT示肠管扩张+液气平面** → 肠梗阻 → **区分SBO vs LBO、机械性 vs 麻痹性、单纯性 vs 绞窄性** → 无绞窄征象→保守治疗（禁食+NGT+补液）24-72h → 绞窄/保守失败 → 急诊手术。

## 核心要点

### 小肠梗阻（SBO）

- **最常见病因**：术后粘连（60-75%）、疝（15-20%）、Crohn病、肿瘤
- **CT（金标准）**：小肠扩张>2.5cm+结肠塌陷→移行区→判断病因
- **保守治疗（无绞窄）**：
  - 禁食+NGT胃肠减压（减少腹胀和误吸）、LR补液纠正容量缺失+电解质
  - **水溶性造影剂挑战**（Gastrografin 50-100mL经NG注入→腹部平片4-8h后）→ 造影剂到达结肠 = 梗阻可自行缓解（保守成功率>90%，住院日↓）
  - 保守观察**最多72h**——超72h不缓解→手术
- **手术指征（"Never let the sun set on a bowel obstruction"原则）**：
  - 绞窄征象（持续剧痛+腹膜炎/反跳痛+代谢性酸中毒+乳酸↑）→ **急诊手术（<6h）**
  - 完全梗阻+无移行区+保守72h不缓解 → 限期手术

### 大肠梗阻（LBO）

- **最常见病因**：结直肠癌（60-70%）、乙状结肠扭转（15-20%）、憩室炎
- **乙状结肠扭转**：CT示"咖啡豆征"+鸟嘴征 → 乙状结肠镜减压（无腹膜炎时成功率~80%）→ 择期乙状结肠切除
- **癌性梗阻**：可切除→一期切除±造口；不可切除→支架置入/造口分流

### 手术紧迫度判断

| 征象 | 意义 |
|------|------|
| **腹膜刺激征（反跳痛/板状腹）** | **绞窄/穿孔 → 急诊手术** |
| **乳酸↑+代酸** | 肠缺血 → 急诊手术 |
| 持续剧痛（非绞痛） | 绞窄——疼痛性质改变=危险信号 |
| CT：肠壁强化减弱/积气/肠系膜水肿** | 缺血/坏死 → 手术 |
| CT：闭环梗阻（Closed loop） | 外科急症——两端均梗阻无法自行缓解 → 限期手术 |

### 保守治疗要素

| 措施 | 细节 |
|------|------|
| **NGT减压** | 持续低负压吸引。盐水冲洗q4-6h。引流量监测——减少提示梗阻缓解 |
| **液体复苏** | LR优先。根据电解质+酸碱调整。K⁺低补钾（低钾加重肠麻痹） |
| **电解质纠正** | 低钾、低镁、低磷都加重肠麻痹——必须纠正 |
| **阿片类药物最小化** | 加重肠麻痹 |
| **无需常规抗生素** | 仅发热/WBC↑/腹膜炎/缺血坏死时启用（哌拉西林他唑巴坦/头孢曲松+甲硝唑） |
