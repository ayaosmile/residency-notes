# 急性上消化道出血的评估与处理

## 结论

呕血/黑便/便血 → **ABC评估 + 开通2条大静脉通路 + 液体复苏** → 危险分层（Glasgow-Blatchford评分）→ **高危患者24h内急诊内镜** → 内镜前PPI（不延迟内镜）+ 内镜下止血。

## 核心要点

- **GBS 评分（Glasgow-Blatchford Score）**：Hb + BUN + SBP + HR + 黑便/晕厥/心衰/肝病。评分 0-1 分 = 低危 → 可门诊；≥ 2 分 → 住院内镜
- **液体复苏**：晶体液 500 mL 快速静滴 → 评估反应。限制性复苏策略（SBP 80-100 mmHg 即可），尤其静脉曲张出血患者避免过度扩容（加重出血）
- **输血指征**：限制性输血策略——Hb < 70 g/L 启动输血，目标 Hb 70-90 g/L（静脉曲张患者 Hb 目标 70-80 g/L 更保守）
- **PPI（质子泵抑制剂）**：内镜前大剂量 PPI——奥美拉唑 80 mg IV bolus → 8 mg/h 持续输注（不延迟内镜，但可降低内镜下出血征象）
- **内镜时机**：高危 → 24h 内；血流动力学不稳定 → 12h 内急诊内镜。**非静脉曲张出血**：Forrest分级决定内镜下治疗（Ⅰa/Ⅰb/Ⅱa → 内镜下止血）
- **静脉曲张出血**：特利加压素/生长抑素 + 预防性抗生素（头孢曲松 1g/d × 7d）+ 12h 内内镜下套扎（EVL）
- **抗凝/抗血小板患者**：华法林→Vit K + PCC（INR > 2.5）；DOAC→停药+必要时拮抗剂；阿司匹林→心血管一级预防停药，二级预防 3-5 天后恢复
