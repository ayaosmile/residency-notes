# 甲状腺功能解读要点

## 结论

甲功五项 → **先看 TSH（最敏感！）** → TSH↓ = 甲亢方向（原发/继发）；TSH↑ = 甲减方向（原发/继发）→ FT3/FT4 确定程度 + **抗体（TPOAb/TgAb/TRAb）确定病因**。

## 核心要点

### 参考范围与筛查流程

| 项目 | 参考范围 | 说明 |
|------|---------|------|
| TSH | 0.27-4.2 mIU/L | **最敏感指标**（对数-线性关系，FT4微小变化→TSH大幅变化） |
| FT4 | 12-22 pmol/L | 甲状腺主要分泌产物 |
| FT3 | 3.1-6.8 pmol/L | 活性形式（FT4 → FT3 外周转化） |
| TPOAb | < 34 IU/mL | 桥本甲状腺炎标志（阳性率 > 90%） |
| TgAb | < 115 IU/mL | 甲状腺球蛋白抗体 |
| TRAb | < 1.75 IU/L | Graves病标志 |

### TSH 为中心的六种模式

| TSH | FT4 | FT3 | 诊断方向 |
|-----|-----|-----|---------|
| ↑ | ↓ | ↓ | **原发性甲减**（最常见——桥本） |
| ↑ | 正常 | 正常 | **亚临床甲减** → TPOAb+ / TSH > 10 或 妊娠 / 有症状 → 启动治疗 |
| ↓ | ↑ | ↑ | **原发性甲亢**（Graves/毒性结节/甲状腺炎） |
| ↓ | 正常 | 正常 | **亚临床甲亢** → 老年/AF/骨质疏松 → 考虑治疗 |
| ↓/N | ↓ | ↓ | **继发性甲减**（垂体/下丘脑）→ MRI排查 |
| 正常 | ↑ | ↑ | **甲状腺激素抵抗**（罕见）或 检测误差/结合蛋白异常 |

### 关键临床场景

**"病态甲状腺功能正常综合征"（NTIS/低T3综合征）**
- 重症/禁食/创伤 → TSH正常，T3↓，T4正常/↓（外周T4→T3转化受抑）
- **不需要甲状腺素替代治疗！** 这是身体的保护机制，治疗可能有害

**妊娠期甲功**
- 前三个月 TSH 参考上限定为 2.5 mIU/L（因 hCG 有 TSH 样作用→TSH 生理性↓）
- 已知 TPOAb+ 的孕妇 → 需监测 TSH 4 周一次 → TSH > 2.5 + TPOAb+ → 启动 LT4 治疗

**亚临床甲减治疗指征**
1. TSH > 10 mIU/L → 治疗（心血管+甲减进展风险）
2. TSH 4.5-10 + TPOAb+ → 治疗
3. TSH 4.5-10 + TPOAb- + 症状/心血管风险 → 个体化
4. 妊娠 TSH > 2.5 + TPOAb+ → **必须治疗**

### 药物影响 TSH

- **激素**：糖皮质激素 → TSH↓
- **多巴胺**：TSH↓
- **胺碘酮**：含高碘 → 甲亢（Ⅰ型）或 甲减（Ⅱ型破坏性甲状腺炎）。用药前和每6个月查甲功
- **锂剂**：抑制甲状腺激素释放 → 甲减
