# 血糖与HbA1c解读要点

## 结论

血糖 = 即时血糖 + **HbA1c（近 2-3 个月平均血糖）**——诊断糖尿病看 FBG + OGTT 2h + HbA1c 三项满足其一即可，糖尿病前期（新增）用 **HbA1c 5.7-6.4%** 协助筛查。

## 核心要点

### 糖尿病诊断标准（中国2型糖尿病防治指南 2024）

| 指标 | 正常 | 糖尿病前期 | 糖尿病 |
|------|------|-----------|--------|
| **空腹血糖（FBG）** | < 5.6 mmol/L | 5.6-6.9 (IFG) | ≥ 7.0 mmol/L |
| **OGTT 2h** | < 7.8 mmol/L | 7.8-11.0 (IGT) | ≥ 11.1 mmol/L |
| **HbA1c** | < 5.7% | **5.7-6.4%（新增）** | ≥ 6.5% |
| **随机血糖 + 症状** | — | — | ≥ 11.1 mmol/L |

> 糖尿病诊断需满足 FBG / OGTT 2h / HbA1c **三项中任一项**（有典型症状时一次即可；无症状需不同日重复确认）

### HbA1c 解读

- **HbA1c** = 糖化血红蛋白，反映近 2-3 个月平均血糖水平（红细胞寿命 ~120 天，近 30 天占 ~50% 权重）
- **估算平均血糖（eAG）**：eAG (mmol/L) = 1.59 × HbA1c - 2.59
  - HbA1c 6.0% ≈ eAG 7.0 mmol/L
  - HbA1c 7.0% ≈ eAG 8.6 mmol/L
  - HbA1c 8.0% ≈ eAG 10.2 mmol/L
- **HbA1c 局限性**（不能替代即时血糖！）：
  - **贫血**（缺铁→假性↑；溶血/近期输血→假性↓）
  - **Hb变异体**（HbS/HbC/HbE）→ HbA1c 不准，用果糖胺或连续血糖监测替代
  - **CKD 4-5期**→ HbA1c 低估（因EPO使用、红细胞寿命缩短）
  - **妊娠**→ HbA1c 生理性↓，不推荐单独用于GDM诊断

### OGTT（口服葡萄糖耐量试验）

- **方法**：空腹 8-14h → 测空腹血糖 → 75g 无水葡萄糖溶于 250-300 mL 水 5 min 内喝完 → 2h 后测血糖
- **GDM（妊娠期糖尿病）**：24-28 周 OGTT（一步法 75g）：
  - 空腹 ≥ 5.1 或 1h ≥ 10.0 或 2h ≥ 8.5 mmol/L → 任意一项超标即诊断 GDM

### 血糖控制目标

| 人群 | 空腹目标 | 餐后2h目标 | HbA1c目标 |
|------|---------|-----------|----------|
| 一般成人 | 4.4-7.0 | < 10.0 | < 7.0% |
| 年轻/病程短/无并发症 | 4.4-6.1 | < 7.8 | < 6.5% |
| 老年/合并症多/CVD病史 | 7.0-8.5 | < 11.1 | < 8.0% |
