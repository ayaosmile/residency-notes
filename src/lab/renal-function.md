# 肾功能解读要点

## 结论

肾功能 = **Cr（快速）+ eGFR（精准）+ 尿量** → Cr 急性升高 → 区分 AKI（KDIGO标准：Cr 48h 内 ↑ ≥ 26.5 μmol/L 或 7d 内 ↑ ≥ 1.5 倍基线）vs CKD基础→ **BUN/Cr 比值**区分肾前性 vs 肾性。

## 核心要点

### 关键指标与参考范围

| 指标 | 男 | 女 | 单位 |
|------|-----|-----|------|
| 血肌酐（Cr） | 57-97 | 41-73 | μmol/L |
| 尿素（BUN） | 3.1-8.0 | 2.6-7.5 | mmol/L |
| 胱抑素C（CysC） | 0.6-1.2 | 0.6-1.2 | mg/L |
| 尿酸（UA） | 208-428 | 155-357 | μmol/L |

### AKI 诊断（KDIGO 2024）

- **Stage 1**：Cr ↑ ≥ 26.5 μmol/L (48h) 或 Cr ↑ 1.5-1.9×基线 (7d) 或 尿量 < 0.5 mL/kg/h × 6h
- **Stage 2**：Cr ↑ 2.0-2.9×基线 或 尿量 < 0.5 mL/kg/h × 12h
- **Stage 3**：Cr ↑ ≥ 3.0×基线 或 Cr ≥ 353.6 μmol/L 或 需要 RRT 或 尿量 < 0.3 mL/kg/h × 24h 或 无尿 × 12h

### eGFR 解读（2021 CKD-EPI 公式，肌酐+胱抑素C联合，已取消种族校正）

- **G1**：≥ 90（正常或升高）
- **G2**：60-89（轻度下降）
- **G3a**：45-59（轻-中度下降）
- **G3b**：30-44（中-重度下降）
- **G4**：15-29（重度下降）
- **G5**：< 15（肾衰竭）→ **eGFR < 60 持续 ≥ 3个月 = CKD**

### BUN/Cr 比值鉴别

- **> 20:1** → 肾前性（脱水、心衰、失血、低灌注），BUN 不成比例升高
- **10-15:1** → 肾性（ATN、AIN、肾小球肾炎）
- **< 10:1** → 肾后性？肝病（BUN 合成减少）？横纹肌溶解（Cr 大幅升高）？

### 危急值

**Cr > 442 μmol/L**（约 eGFR < 12）→ 肾内科紧急会诊。**K⁺ > 6.5 mmol/L + Cr 极度升高** → 紧急透析指征。
