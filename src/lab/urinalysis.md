# 尿常规解读要点

## 结论

尿常规 = **干化学（试纸法）+ 尿沉渣镜检（金标准）** → 按"蛋白-潜血-白细胞-糖-酮体"五大要素逐一判读 → **干化学阳性必须镜检确认**。

## 核心要点

### 参考范围（WS/T 229-2024, 2024年11月实施）

| 项目 | 正常值 |
|------|--------|
| pH | 4.5-8.0 |
| 比重（SG） | 1.015-1.025 |
| 蛋白质（PRO） | 阴性（-） |
| 葡萄糖（GLU） | 阴性（-） |
| 酮体（KET） | 阴性（-） |
| 潜血（BLD） | 阴性（-） |
| 亚硝酸盐（NIT） | 阴性（-） |
| 白细胞酯酶（LEU） | 阴性（-） |
| 胆红素/尿胆原 | 阴性/正常 |
| RBC（镜检） | < 3/HP |
| WBC（镜检） | < 5/HP |
| 透明管型 | 偶见（≤ 1/LP） |

### 关键判读

**蛋白尿（PRO+）**
- **生理性**：发热、剧烈运动、直立性（青少年晨尿阴性、下午尿阳性）、妊娠
- **病理性**：肾小球（肾小球肾炎、糖尿病肾病）→ 尿蛋白电泳；肾小管（间质性肾炎、Fanconi综合征）→ 低分子量蛋白（β2-MG/α1-MG）
- **微量白蛋白尿**：30-300 mg/24h 或 ACR 3-30 mg/mmol → **糖尿病肾病早期标志**，可逆转期

**血尿（BLD+）**
- 试纸阳性 + 镜检未见 RBC = **肌红蛋白尿**（横纹肌溶解）或 **血红蛋白尿**（溶血）
- 试纸阳性 + 镜检 RBC > 3/HP = 真性血尿 → 区分肾小球性（变形RBC > 80%、RBC管型、蛋白尿++）vs 非肾小球性（均一RBC、血块、无蛋白尿）
- 无症状镜下血尿（无蛋白尿、GFR正常）→ 建议影像学排查（超声+膀胱镜）

**白细胞尿（LEU+/WBC↑）**
- **UTI 三联征**：LEU+ + NIT+ + WBC > 10/HP → 尿路感染（中段尿培养金标准）
- 无菌性白细胞尿：SLE、间质性肾炎、前列腺炎、结核

**管型**
- **透明管型**：偶见正常；增多 → 脱水、利尿剂、运动后
- **颗粒管型**：ATN、慢性肾脏病
- **RBC管型** → **肾小球肾炎（高度特异！）**——红细胞来自肾小球
- **WBC管型** → 间质性肾炎、肾盂肾炎
- **蜡样管型** → 慢性肾衰竭晚期
