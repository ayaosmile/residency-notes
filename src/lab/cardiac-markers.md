# 心肌标志物解读要点

## 结论

胸痛 → **hs-cTn（高敏肌钙蛋白）是"金标准"**：0h/1h 或 0h/3h 快速排除/纳入方案 → CK-MB 辅助判断再梗死和梗死时间 → **BNP/NT-proBNP 区分心源性呼吸困难**。

## 核心要点

### hs-cTn（高敏肌钙蛋白）——AMI金标准

| 方案 | 排除标准 | 纳入标准 | 观察窗 |
|------|---------|---------|--------|
| **ESC 0h/1h** | 0h < 3 ng/L + Δ1h < 2 ng/L | 0h ≥ 52 ng/L 或 Δ1h ≥ 5 ng/L | 中间值观察3h |
| **ESC 0h/3h** | 0h < 3 ng/L + Δ3h < 3 ng/L | 0h ≥ 52 ng/L 或 Δ3h ≥ 7 ng/L | 中间值观察6h |

> **注意**：以上阈值为 hs-cTnT (Roche) 参考值，**各厂商阈值不同，以本地实验室 99 百分位值为准**。hs-cTnI 使用不同 cutoff。

- **非 AMI 原因肌钙蛋白升高**（肌钙蛋白 = 心肌损伤，不是心梗的代名词！）：心衰、肺栓塞、心肌炎、肾衰（CKD4-5期持续性轻度升高）、房颤快速心室率、Takotsubo、脓毒症、剧烈运动（马拉松后）
- **升高模式**：AMI → 2-4h 起升 → 12-24h 达峰 → 持续 7-14 天（缓慢下降）。**CK-MB 用于判断再梗死**（hs-cTn持续高值时，CK-MB 已回落→再升 = 再梗死）

### CK-MB

- 参考范围：< 25 U/L（或 < 3-5 μg/L，依方法学）
- **AMI**：4-6h ↑ → 12-24h 达峰 → 48-72h 降至正常
- **再梗死判断**：CK-MB 已下降 > 50% 后再次升高 → 新发梗死。hs-cTn 因下降缓慢难以单独判断
- **非心源性**：横纹肌溶解（CK-MM 主导，CK-MB/CK 比值 < 5%）、心肌炎

### BNP / NT-proBNP

| 指标 | 排除急性心衰 | 诊断急性心衰 | 灰色地带 |
|------|------------|------------|---------|
| **BNP** | < 100 pg/mL | > 300 pg/mL | 100-300 |
| **NT-proBNP** | < 300 pg/mL | 年龄分层见下 | 300-年龄诊断值 |

**NT-proBNP 年龄分层诊断值**（ESC 2021/2023共识）：
- < 50 岁 → > 450 pg/mL
- 50-75 岁 → > 900 pg/mL
- > 75 岁 → > 1800 pg/mL

- **影响 NT-proBNP 的因素**：
  - 升高：肾衰（清除减少，CKD4-5期常规偏高）、高龄、房颤、肺栓塞、肺动脉高压
  - 降低：肥胖（BNP 被脂肪组织降解，BMI > 35 诊断阈值需下调 ~50%）
