# 血常规解读要点

## 结论

血常规 = **三系**（白细胞 + 血红蛋白/红细胞 + 血小板）→ 逐一判读 → 异常按方向排查（增多/减少）→ 结合临床背景给出鉴别。

## 核心要点

### 白细胞（WBC）

- **WBC ↑**：感染（最常见）、应激、激素、血液病。看中性粒细胞 vs 淋巴细胞比例区分细菌 vs 病毒
- **WBC ↓**：病毒感染、药物（化疗/抗甲状腺药）、再障、SLE
- **中性粒细胞绝对计数 < 0.5×10⁹/L** → 粒缺，发热即按粒缺伴发热处理（立即血培养 + 经验性广谱抗生素）
- **淋巴细胞 ↑**：病毒感染（EBV/CMV）、结核、百日咳、CLL

### 血红蛋白（Hb）

- **贫血判读**：男 < 130 g/L，女 < 120 g/L，孕妇 < 110 g/L
- **MCV（关键分型指标）**：< 80 fL（小细胞：缺铁/地贫/慢性病）；80-100 fL（正细胞：急性失血/慢性病/肾性贫血）；> 100 fL（大细胞：B12/叶酸缺乏、酒精、MDS）
- **急性 Hb 下降**：每下降 10 g/L ≈ 失血 400-500 mL

### 血小板（PLT）

- **PLT < 50×10⁹/L**：手术/操作禁忌
- **PLT < 20×10⁹/L**：自发性出血风险
- **PLT < 10×10⁹/L**：颅内出血风险，需输注血小板
- **PLT ↓**：ITP、DIC、药物、脾亢、肝硬化、再障
- **PLT ↑**：感染/炎症反应、缺铁、脾切除后、MPN

### 全血细胞减少（三系↓）

再障、MDS、急性白血病、脾亢、SLE、药物。
