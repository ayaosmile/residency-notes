# 电解质解读要点

## 结论

电解质七项 → **先看 K⁺（危及生命的心律失常）+ Na⁺（神经/容量状态）+ Ca²⁺（必需校正白蛋白！）** → 按异常方向（高/低）结合临床排查 → 危急值立即处理。

## 核心要点

### 参考范围（WS/T 404.3-2012, WS/T 404.6-2015）

| 项目 | 参考范围 | 危急值 |
|------|---------|--------|
| K⁺ | 3.5-5.3 mmol/L | **< 2.5 或 > 6.0** |
| Na⁺ | 137-147 mmol/L | **< 120 或 > 160** |
| Cl⁻ | 99-110 mmol/L | — |
| 总 Ca²⁺ | 2.11-2.52 mmol/L | < 1.8 或 > 3.5 |
| Mg²⁺ | 0.75-1.02 mmol/L | < 0.5 |
| 无机磷 | 0.85-1.51 mmol/L | — |

### 必须记住的公式

**白蛋白校正钙**：校正总钙 = 实测总钙 - 0.025 × 白蛋白 + 1.0

> 白蛋白每下降 10 g/L，总钙约低 0.25 mmol/L。低白蛋白时不校正钙会导致漏诊真正的高钙血症。

### 低钠血症诊断思路

1. **测血浆渗透压**（排除假性低钠：高血糖/高脂血症/高蛋白血症）
2. **等渗低钠**（280-295 mOsm/kg）→ 假性
3. **高渗低钠**（> 295）→ 高血糖（血糖每 ↑ 5.5 mmol/L，Na⁺ ↓ 1.6 mmol/L）
4. **低渗低钠**（< 280）→ 评估容量（低容/等容/高容）→ 尿钠区分
   - 尿钠 < 20 mmol/L → 肾外丢失（呕吐/腹泻/心衰/肝硬化）
   - 尿钠 > 40 mmol/L → 肾性丢失（利尿剂/盐耗/脑耗盐综合征/SIADH）

### 阴离子间隙（AG）

AG = Na⁺ - (Cl⁻ + HCO₃⁻)，正常 8-12 mmol/L
- **高AG酸中毒**：酮症(DKA)/乳酸/肾衰/中毒（甲醇/乙二醇/水杨酸）
- **正常AG酸中毒**：腹泻/肾小管酸中毒/输尿管乙状结肠吻合——HCO₃⁻ 丢失+Cl⁻ 代偿性升高
