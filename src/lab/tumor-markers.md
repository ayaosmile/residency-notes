# 肿瘤标志物解读要点

## 结论

肿瘤标志物 **不能用于无症状人群的癌症筛查！** 核心价值在于：①已知恶性肿瘤的**疗效评价和复发监测**（同一标志物连续监测）；②临床高度怀疑时辅助诊断。单项升高 ≠ 癌症——良性病变也可升高。

## 核心要点

### 常用肿瘤标志物一览

| 标志物 | 主要相关肿瘤 | 良性升高原因 |
|--------|------------|------------|
| **CEA** | 结直肠癌、胃癌、肺癌、胰腺癌 | 吸烟、炎症性肠病、肝硬化、胰腺炎、COPD |
| **AFP** | 原发性肝癌（HCC）、卵黄囊瘤 | 肝炎/肝硬化活动期、妊娠 |
| **CA19-9** | 胰腺癌、胆管癌、胃癌 | 胆道梗阻/炎症、胰腺炎、肝硬化（Lewis抗原阴性者完全不表达） |
| **CA125** | 卵巢癌（上皮性） | 盆腔炎、子宫内膜异位、子宫肌瘤、妊娠、月经期、腹水（任何原因） |
| **CA15-3** | 乳腺癌（转移监测） | 肝硬化、良性乳腺疾病、盆腔炎 |
| **PSA** | 前列腺癌 | 前列腺炎、BPH、排尿困难、骑自行车、性活动后 |
| **NSE** | 小细胞肺癌（SCLC）、神经内分泌瘤 | 溶血（标本不合格！）、机械瓣溶血 |


### 临床使用的核心原则

1. **不用于无症状筛查**（除 PSA 有争议外）——低阳性预测值 + 假阳性带来不必要的侵入性检查和焦虑
2. **术前升高 = 术后监测标志**。术后标志物应降至正常（半衰期：CEA 3-5 天；CA19-9 数天-数周）。术后不降或降后又升 = 残留/复发
3. **AFP ≥ 400 μg/L + 肝硬化 + 影像学占位** → HCC 诊断（该组合特异性 > 95%）
4. **CA19-9 在 Lewis 抗原阴性者（~7% 人群）中根本不产生** → 此类患者不能用 CA19-9 监测
5. **PSA 筛查**：50-69 岁男性经充分知情同意后可考虑（U.S. PSTF Grade C），但过度诊断和过度治疗争议持续

### 解读注意事项

- **"参考范围正常"不排除癌症**——早期癌症标志物往往正常
- **"轻度升高"不要首先想到癌症**——排除良性原因（复查、问病史、查影像）→ 如找不到良性原因 → 定期随访观察趋势
- **动态趋势优于单次值**——连续升高 > 单次升高 > 绝对值
- **NSE 标本**：溶血导致假性升高（血小板和红细胞含 NSE），不合格标本必须拒收
