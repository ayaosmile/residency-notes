# 交接班与SBAR沟通法

## 结论

SBAR（**S**ituation-**B**ackground-**A**ssessment-**R**ecommendation）= 标准化医疗信息沟通框架。用于交接班、汇报上级、跨科室转诊/会诊沟通。目标：**30-60秒内让对方抓住关键信息**。

## SBAR 四要素

### S — Situation（现状）
**"我是谁，我现在担心什么"**
- 你的身份+科室
- 患者姓名、年龄、床号
- **当前最紧急问题**：一句话说清"我为什么要和你沟通"

> 示例："我是内科值班李医生。3床 45岁男性，刚刚开始大量咯血，5分钟内咯血量超过200mL。"

### B — Background（背景）
**"你知道这些背景就能理解现在的问题"**
- 主要诊断+入院日期
- 相关既往史（与该问题直接相关的，不罗列所有）
- 近期关键干预/手术/用药变化

> 示例："肺炎入院第3天，既往COPD/吸烟40包年。昨天开始普通病房治疗，口服抗生素+止咳。今晨突然加重。"

### A — Assessment（评估）
**"我认为问题是什么"**
- 生命体征
- 关键查体发现
- 检验/影像关键结果
- **你的判断**："我认为这可能是XX"

> 示例："SaO₂ 88%尽管15L面罩给氧，BP 90/60，HR 130。呼吸音右中下叶消失。CXR显示右中叶气液平+大片浸润近肺门。我认为可能是坏死性肺炎合并支气管胸膜瘘导致大咯血。"

### R — Recommendation（建议）
**"我需要你做什么/我建议下一步怎么办"**
- 明确、具体的要求或下一步建议
- 时间期望（现在？30分钟内？24小时内？）

> 示例："我需要胸外科立即来评估+介入科评估支气管动脉栓塞。我已经准备了CT增强。希望30分钟内胸外到位。"

## 交接班专用：SBAR + 待办清单

交接班时在SBAR后附加：

- **夜间预期**："预计今晚最可能出问题的地方是……"
- **触发呼叫标准**："如果出现 XX（BP<90/尿量<30ml/h/SpO₂<88%），立即联系我"
- **待办事项清单**：具体到什么时间谁做什么

## 常见沟通错误

| 错误 | 纠正 |
|------|------|
| "X床情况不好" | 说清哪个指标、什么趋势、多危急（SBAR的A） |
| "需要看一下" | 说清需要什么行动、多紧急（SBAR的R） |
| 先说背景再说紧急 | **紧急问题先说**（SBAR的S始终第一） |
| 把全部病史复读一遍 | 只说与当前问题相关的关键背景 |
