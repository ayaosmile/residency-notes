# 知情同意谈话要点

## 结论

知情同意 = **诊断→治疗方案→获益→风险→替代方案→不治疗的后果** 六大要素缺一不可。核心是"患者/家属充分理解后自愿同意"，不是"签字免责"。

## 六大必谈要素

1. **当前诊断/病情**：用患者听得懂的语言，不堆医学术语。"心脏的血管堵了"不是"大隐静脉桥血管闭塞"
2. **拟施行的治疗方案**：具体操作/手术/用药方案、谁来执行（主治+住院医？单独操作？）
3. **预期获益**：诚实的成功率和好转可能性——不夸大。"缓解疼痛"不等同于"完全恢复"
4. **主要风险**：
   - **常见但轻微**（术后疼痛/恶心/疤痕——几乎每个人都会经历）
   - **少见但严重**（感染/出血/栓塞/死亡——发生率用"百分之几"而不用"罕见/低"）
   - 注意：风险必须包括**死亡风险**（任何有创操作/全麻都应谈）
5. **替代方案**（有或无）：列举其他治疗选项——包括"选择不治疗"。
6. **拒绝治疗的可能后果**：病情自然进展/加重的预测后果

## 关键原则

- **谈话对象优先顺序**：患者本人（自主决定权）→ 授权委托人（授权委托书）→ 近亲属（按配偶→成年子女→父母→兄弟姐妹→祖父母/成年孙子女顺序）
- **紧急情况例外（"急诊救治原则"）**：危及生命且无法取得本人/家属意见 → 经医疗机构负责人或授权负责人批准 → 可立即实施救治。事后补办
- **特殊人群**：
  - 未成年人：父母/法定监护人签字（≥8周岁儿童应参与决策——未成年人保护法精神）
  - 意识/精神障碍但非紧急：鉴定+法定代理人
  - 文化/宗教冲突（如耶和华见证会拒绝输血）：应书面记录其特定拒绝 + 告知替代方案
- **记录**：谈话时间、参与人（姓名+与患者关系）、谈话内容要点、理解确认、签字+按手印。**只签一个名字不算有效知情同意**

### 医患沟通"四不"

1. **不做100%承诺**——"手术很安全"（错）→ "手术成功率约X%，大约每100个人中有Y个人可能发生Z并发症"（对）
2. **不贬低其他医生/医院**——法律责任+医德双失分
3. **不替患者/家属做决定**——你的角色是提供信息+建议，不是代替签字
4. **不在非紧急时跳过正规流程**——一旦出纠纷，程序不合法 → 无处可辩
