# 病危/病重通知与谈话

## 结论

病危/病重通知 = **第一时间告知 + 诊断→现状→即刻风险→正进行/计划中的救治措施→预后不确定性** 五大要素。不能只说"病危了签个字"——必须讲清病情严重在哪里、你在做什么、接下来可能发生什么。

## 通知五步法（SPIKE模型）

### 1. Setting（准备环境）
- 安静、隐私空间（值班室/洽谈室，不在走廊/病房多人旁谈）
- 关键家属尽可能到场（或电话+扩音同时告知）
- 坐姿（平视或医生低于家属视线）+ 眼神接触

### 2. Perception（了解家属认知）
- **先问**："你对XX的病情目前了解多少？"
- 了解家属对病情严重程度的基线认知。如果与实际情况落差大，需要更多铺垫

### 3. Invitation（告知准备）
- "XX的病情目前很危重/严重，我需要和你说一下具体情况和我们现在正在做的事情"
- 用 **"危重"** 或 **"严重"** 直接开头——为后续不利信息建立预期

### 4. Knowledge（传递信息）
- **诊断现状**：用通俗语言解释最严重问题。"他的血压需要用最大剂量的升压药维持，这是一个非常危险的信号"
- **即刻风险**：最直接的生命威胁是什么——不要回避"死亡"二字（家属需要真实的信息来做决定）
- **正在做的抢救/治疗**：列出在运行的抢救措施——让家属知道你是全力的，保护感和信任感由此建立
- **不确定性**：诚实交代你不知道的——"接下来24-48小时是最关键的时刻，我们会持续监测和调整。现在无法确定他能否渡过这个难关"

### 5. Empathy（情感回应）
- 停顿，给家属消化和提问的空间
- "我知道这很难接受" 或 "有任何想了解的都可以问我"
- 沉默不比硬塞安慰差——安静的陪伴有时是最有力的共情

## 高频场景要点

### 终末期/临终谈话
- "是不是没希望了？" → 如确实无逆转可能 → 诚实：**"目前的所有治疗已经无法逆转他的病情，我们现在的重点转向让他不那么痛苦地走完最后一程"**
- 避免"我们也没什么办法了"——永远有办法：镇痛、镇静、保暖、陪伴、尊严
- 安宁疗护/间停积极治疗的选项应在此时提出

### 急诊/抢救中家属要求入室
- 原则上抢救室不应有家属在场（干扰操作+心理创伤+医疗纠纷风险）
- 但可以：指派专人（非抢救操作人员）间歇向家属通报进展（每15-30min）
