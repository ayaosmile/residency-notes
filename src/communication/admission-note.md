# 入院记录书写规范

## 结论

入院记录 = **主诉（症状+时间）+ 现病史（"7要素"）+ 既往史 + 体格检查 + 辅助检查 + 初步诊断 + 诊疗计划**。核心原则：**客观描述、时间顺序、逻辑推理、鉴别诊断思路**。

## 七大部分要点

### 1. 主诉
- **格式**：主要症状/体征 + 持续时间（≤20字）
- "胸痛3小时" "发热、咳嗽5天" "发现血压升高2年"
- 用患者自己的描述，不做诊断性改写

### 2. 现病史（核心中的核心）
- **7要素**：①起病情况+诱因 ②主要症状特点（部位/性质/持续/缓解加重因素）③伴随症状（有/无——阴性症状同样重要）④病情演变 ⑤诊疗经过（外院+本院——药物+剂量+效果）⑥发病以来一般情况（食欲/睡眠/二便/体重）⑦鉴别诊断相关的阴性症状（必须写！）
- **时间轴**：从最早症状按时间顺序叙述至今

### 3. 既往史
- 疾病史（高血压/糖尿病/心脏病/肝病/肾病/COPD/肝炎/结核）
- 手术外伤史（名称+年份）
- 输血史、过敏史（药物/食物）
- 预防接种史（新冠/流感/肺炎球菌/乙肝等）
- **月经婚育史**（女性）
- **药物史**：现服所有药物（名称+剂量+频次+适应症）

### 4. 体格检查
- **生命体征**：T/HR/BP/RR/SpO₂
- 系统性全身体检（皮肤/头颈/胸/腹/脊柱四肢/神经系统）
- 专科查体（病变系统做详细检查）

### 5. 辅助检查
- 门急诊/外院已做的检查结果——注明检查日期+机构
- 重要的阴性结果也列出

### 6. 初步诊断
- **主次分明**：主要诊断 → 并发症 → 伴发病
- 不确定时："XX待查（鉴别诊断A/B/C）"比假确诊更好
- 用诊断标准+编码

### 7. 诊疗计划
- **诊断计划**：还需做什么明确/排除诊断？
- **治疗计划**：具体方案（药物+剂量+频次+疗程）
- **谈话计划**：是否需知情同意谈话+通知家属
- **监测计划**：什么指标、多久查一次

## 法律注意

- 入院记录应在 **24小时内** 完成
- **修改**：划线修改+签名+日期（不能用涂改液/刀片刮/全擦）。电子病历修改 = 留痕迹
- **不可以**：模仿/伪造患者或家属签名、补造缺失记录、倒填时间

## 来源

- 病历书写基本规范（2024版，国家卫生健康委）
- 医疗机构病历管理规定（2023修订）
- 中国病案管理协会病历质量评分标准（2024）
