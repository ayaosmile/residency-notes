# 胸腔闭式引流术

## 结论

胸腔闭式引流 = **气胸/胸腔积液/血胸/脓胸大量** → 超声/CT定位 → 安全三角区（腋中线第5肋间）→ 尖钳分离肋间肌+手指探查胸膜腔 → 胸管置入+水封瓶连接 → 出现水柱波动/气泡确认。

## 核心要点

- **安全三角区（Safe Triangle）**：腋中线第5肋间——前界胸大肌外侧缘、后界背阔肌前缘、下界乳头连线（第5肋水平）。此三角内肌肉薄、血管少
- **管径选择**：气胸→18-22F；胸腔积液→20-24F；血胸→28-32F；脓胸→28-32F。小管径在积液/血胸时易堵塞
- **穿刺技术**（逐层分离——避免直接粗暴穿刺）：
  1. 利多卡因浸润至壁层胸膜（回抽有气体/液体 = 确认在胸膜腔）
  2. 皮肤切口 1.5-2cm
  3. 尖钳钝性分离肋间肌至壁层胸膜 → **手指伸入胸膜腔探查有无粘连**
  4. 沿手指引导送入胸管 → 尖端朝向肺尖（气胸）或后肋膈角（积液）
- **连接水封瓶**：水封深度 2-3cm。接触胸腔后立即出现水柱随呼吸波动（吸气↑，呼气↓）= 胸管在胸膜腔
- **术后CXR（必须确认！）**：确认胸管位置正确 + 肺复张状态 + 排除并发症（叶间裂置管/肺内/膈下/皮下）

## 拔管时机

- 气胸：无气泡 > 24h + CXR肺复张 → 夹管4-6h → CXR持续复张 → Valsalva下拔管
- 积液：引流量 < 100-150 mL/24h + 积液几乎消失 + 无感染征象

## 并发症

| 并发症 | 预防 |
|--------|------|
| 管错位（叶间裂/肺内/膈下/皮下） | 手指探查+术中CXR/超声 |
| 复张性肺水肿 | 引流速度控制（首次+连续不宜过快） |
| 出血（肋间血管损伤） | 肋骨上缘分离+安全三角区内操作 |
| 感染（脓胸） | 严格无菌屏障。引流 > 72h → 感染率上升 |
