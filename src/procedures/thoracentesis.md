# 胸腔穿刺术

## 结论

胸腔穿刺 = **超声定位（强制，非盲穿！）** → **坐位/侧卧** → **肋骨上缘进针**（避开肋间血管神经束）→ 诊断性 ≤ 50 mL，引流性首次 ≤ 1500 mL（过快→复张性肺水肿）。

## 核心要点

- **穿刺点**：超声定位积液最深处。传统体表标志 = **肩胛线第 7-8 肋间** 或 **腋后线第 8-9 肋间**（定位需比超声低一个肋间，避免膈肌损伤）
- **肋骨上缘进针（绝对原则！）**：肋间神经/血管走行于肋骨下缘沟 → 肋骨下缘进针→血管损伤→血胸
- **麻醉**：2% 利多卡因 5-10 mL 逐层浸润（皮肤→皮下→骨膜→胸膜）
- **进针阻力消失感**：穿过壁层胸膜时出现 → 停止推进 → 回抽到液体/气体
- **引流量**：
  - 诊断性：20-50 mL 送检（生化+常规+病理+培养+ADA/TB-PCR）
  - 治疗性：首次最大 1500 mL（单侧一次性）。超过 1500 mL → 24h 后可继续引流
- **复张性肺水肿（RPE）预警**：剧烈咳嗽、泡沫痰、SpO₂↓ → 立即停止引流 + 夹管 + 氧疗。发生率随引流速度和量增加
- **禁忌状况**：INR > 2.0 或 PLT < 25×10⁹/L → 先纠正（FFP+PCC/血小板输注）。**唯一例外：张力性气胸**（抢救性针减压不等纠正凝血）

## 胸水送检

- **Light标准**（区分渗出液/漏出液）：
  - 胸水蛋白/血清蛋白 > 0.5 或 胸水LDH/血清LDH > 0.6 或 胸水LDH > 2/3 血清LDH上限 → **渗出液**（满足任一条）
- 渗出液 = 感染/肿瘤/结缔组织病/PE → 继续查ADA、TB-PCR、细胞学、pH
