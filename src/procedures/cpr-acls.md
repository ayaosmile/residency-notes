# 成人心脏骤停与CPR/ACLS

## 结论

确认无反应+无正常呼吸 → **立即启动 EMS + 开始胸外按压（30:2，深度 5-6cm，速率 100-120/min）** → 除颤器到位即刻 → **可电击心律（VT/VF）→ 立即除颤**（非可电击心律→持续CPR+查找病因）→ 每 2 分钟轮换按压 + 肾上腺素 q3-5min。

## 核心要点

### 基础生命支持（BLS）

| 操作 | 要点 |
|------|------|
| 识别 | 无反应 + 无正常呼吸/喘息（agonal breathing 不是正常呼吸）= 心脏骤停 |
| 按压:通气 | **30 次按压 : 2 次通气**（单人/双人均为 30:2） |
| 按压速率 | **100-120/min**（太快→回弹不足；太慢→心输出量不足） |
| 按压深度 | **5-6 cm**（成人）——**完全回弹**（胸廓不回弹→心脏无法充盈 = 无效按压） |
| 按压中断 | **任何中断 ≤ 5-10s**（持续按压时间比例 ≥ 60%） |
| 通气 | 每次用 1s，可见胸廓抬起即停。**避免过度通气！**（增加胸内压→降低心输出量） |
| AED/除颤 | **到位即用——先除颤后CPR（可电击心律：VT/VF）** |

### 高级生命支持（ACLS）

**可电击心律（VT/VF）**
- CPR + **除颤（双相 200J）** → 5个循环CPR（~2min）→ 检查脉搏+心律
- 第2-3次除颤后 → 肾上腺素 1mg IV（q3-5min 重复）
- 第3-5次除颤后 → 胺碘酮 300mg IV bolus；顽固性者 → 150mg IV 重复
- **持续电击+药物+HSM循环直到ROSC或终止复苏**

**非可电击心律（停搏/PEA）**
- CPR + 肾上腺素 1mg IV q3-5min（立即启动）→ 不需要除颤
- 重点：**找到并逆转可逆病因→否则无法获得ROSC**

### 可逆病因 (Hs & Ts)

| 5H | 5T |
|----|-----|
| **H**ypoxia（缺氧） | **T**ension pneumothorax（张力性气胸） |
| **H**ypovolemia（低容量） | **T**amponade（心包填塞） |
| **H**ydrogen ion（酸中毒） | **T**oxins（药物/毒物过量） |
| **H**ypo/Hyperkalemia（低/高钾） | **T**hrombosis - coronary（AMI） |
| **H**ypothermia（低体温） | **T**hrombosis - pulmonary（肺栓塞） |

### 复苏后管理（ROSC后）

- **目标 SpO₂ 94-98%**（避免高氧——自由氧自由基损伤）
- **MAP ≥ 65 mmHg**（去甲肾/液体维持）
- **紧急冠脉造影**（STEMI/新发LBBB/高度怀疑心源性 → 立即PCI——无意识不排除PCI）
- **目标体温管理（TTM）**：32-36°C（至少维持 24h）。无论初始心律是否可电击
- **神经预后判断**：ROSC 后 ≥ 72h（至少代谢药物清洗后）进行评估——瞳孔+角膜反射+N20 SSEP+脑电图+脑MRI 综合判断
