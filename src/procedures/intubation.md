# 气管插管术

## 结论

气管插管 → **准备（7P法）+ 预氧合 3-5min** → **快速顺序诱导（RSI）** 或 清醒插管（困难气道）→ **确认导管位置（ETCO₂+双侧呼吸音+胸廓对称起伏）** → 固定深度（门齿 21-23 cm 女 / 23-25 cm 男）。

## 核心要点

### 插管指征（合并以下任一 = 准备插管）

1. **气道保护丧失**（GCS ≤ 8 / 气道反射丧失 / 误吸）
2. **氧合衰竭**（PaO₂ < 60 或 SpO₂ < 90% 尽管给氧）
3. **通气衰竭**（PaCO₂ > 55 + pH < 7.25 / RR > 35）
4. **预计病情恶化**（烧伤吸入性损伤、颈部血肿扩大、顔面多发骨折）

### "7P" 准备法

1. **P**reparation（评估+药：困难气道评估——mallampati、甲颏距、张口度、颈部活动）
2. **P**reoxygenation（预氧合：100% FiO₂，3-5 min。肺水肿/ARDS→HFNC/NIV预氧合）
3. **P**retreatment（预处理：芬太尼 2-3 μg/kg 镇压插管应激）
4. **P**aralysis with induction（诱导+肌松）：
   - 诱导剂：丙泊酚 1.5-2 mg/kg IV（低血压用依托咪酯 0.3 mg/kg 或 氯胺酮 1-2 mg/kg）
   - 肌松剂：罗库溴铵 1.0 mg/kg（RSI）或 琥珀胆碱 1.5 mg/kg（快起快退，高钾/烧伤/挤压伤禁用）
5. **P**ositioning（嗅花位——sniffing position）
6. **P**ass the tube（Macintosh 直接喉镜 或 可视喉镜 → 看见声门 → 通过导管 → 充气囊 → ETCO₂ 确认）
7. **P**ost-intubation management（ETCO₂ > 0 是金标准 → 双侧呼吸音对称 → 固定 → CXR 确认位置——导管尖端在 T4 水平 = 气管中段，隆突上 3-5cm）

### 确认导管位置

| 方法 | 敏感性 | 特异性 |
|------|--------|--------|
| **ETCO₂（二氧化碳波形图）** | 100% | 100% → **金标准** |
| 双侧呼吸音 | — | 可假性（支气管内插管/气胸有消失侧） |
| 胸廓起伏 | — | 不可靠 |
| CXR | — | 金标准最终确认——**尖端不应在隆突水平以下（误入右主支气管）** |

### 困难气道预案

如果第一次失败 → **面罩通气维持氧合 → 再准备**：
- 可视喉镜（增加首次成功率 10-20%）
- 声门上装置（LMA作为过渡氧合桥）
- 外科气道（环甲膜切开——"can't intubate, can't oxygenate"最后手段）
