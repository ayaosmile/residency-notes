# 腰椎穿刺术

## 结论

腰穿 = **适应证确定（CNS感染/SAH/炎症/肿瘤）→ CT排除占位效应（意识障碍+局灶体征+视乳头水肿时）→ 侧卧位 L3-4/L4-5 进针 → 测开放压 + 留 CSF × 4 管**。

## 核心要点

- **穿刺前 CT？** 仅限存在以下任一 → 先CT再穿：意识障碍、局灶神经体征、新发癫痫、视乳头水肿、免疫抑制——排除占位效应/中线移位（防止脑疝）
- **体位与定位**：侧卧屈膝弓背（脊柱水平面），**L3-4 或 L4-5 棘突间隙**（髂嵴连线 = L4 棘突，这是安全定位标志。脊髓末端终止于 L1-2，更低位 = 仅马尾神经——穿刺安全）
- **进针角度**：针尖斜面朝上（平行脊柱纵轴），向脐方向、头端倾斜 15° → 阻力↓+突破感 → **拔出针芯见 CSF 流出**
- **开放压**：正常 60-200 mmH₂O（侧卧位）。> 200 = 颅压增高（SAH/脑膜炎/假性脑瘤/占位）；< 60 = 低颅压（CSF漏/脱水）
- **留管**：4 管，每管 0.5-2 mL。第1管 → 生化+蛋白+糖（同步血糖！）；第2管 → 革兰染色+细菌培养；第3管 → 细胞计数+分类；第4管 → 细胞学/特殊检查
- **CSF糖/血糖比**：正常 > 0.6。**< 0.4 → 细菌性脑膜炎**（细菌耗糖）。≥ 0.6 但WBC↑ → 病毒性脑膜炎
- **并发症**：
  - **腰穿后头痛（PLPH）**——30%，24-48h内，直立加重，平卧减轻。使用 **无创穿刺针（22G atraumatic needle，Sprotte/Whitacre）** 可将发生率从 36% 降至 3%
  - 硬膜外血肿（< 1/10万）→ 凝血异常是绝对禁忌
  - 脑疝——CT评估占位效应的目的在此，极罕见但致命
