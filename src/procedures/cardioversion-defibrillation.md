# 电复律与电除颤

## 结论

**不稳定的心律失常 → 立即电击（不等药物！）**。同步电复律（SYNC ON）= 直流电击与 QRS 波同步，避免在 T 波上放电诱发 VF。非同步电除颤（SYNC OFF）= VF/无脉 VT，随时放电。

## 核心要点

| | 同步电复律（SYNC ON） | 非同步电除颤（SYNC OFF） |
|---|---|---|
| **适应证** | 不稳定房颤/房扑/AVNRT/SVT/VT(有脉) | **VF、无脉VT、多形性VT(无脉)** |
| **关键！！！** | **必须确认 SYNC ON！** | SYNC OFF |
| **能量(双相波)** | 房颤 200J；房扑/SVT 50-100J；VT(有脉) 100J | **首次 200J（最大功率）** |

### 操作步骤

1. **适应证确认**：不稳定？→ 电击！稳定？→ 暂停，药先
2. **镇静**（仅稳定患者 + 同步复律）：丙泊酚 0.5-1 mg/kg IV 或 咪达唑仑 1-3 mg IV → 达到无意识/无痛反应
3. **电极位置**：**前-侧位（推荐）**——右胸骨旁第2肋间 + 左腋中线第5-6肋间（心尖）。**前-后位**（如需更高成功率，如房颤）——前胸骨左缘 + 后侧左肩胛下方
4. **导电凝胶/导电垫**：必须使用，避免皮肤烧伤+阻抗降低
5. **充电 → 确认所有人离开（"All clear!"）→ 放电**
6. **放电后立即恢复 CPR 2min 不中断**（即使恢复窦律也需要恢复心肌灌注）→ 再检查心律+脉搏
7. **SYNC 复律每击前重新按 SYNC 按钮**——部分除颤器放电后自动退出 SYNC 模式

### 常见心律失常电击方案

| 心律 | 能量 | 同步？ | 备注 |
|------|------|--------|------|
| VF / 无脉 VT | **200J（最大）** | **NO** | 不间断CPR+肾上腺素q3-5min |
| 不稳定房颤 | 200J（双相）/ 200J（单相） | **YES** | SYNC ON必须确认！ |
| 不稳定房扑 | 50-100J | YES | 房扑对低能量更敏感 |
| 不稳定SVT | 50-100J | YES | 腺苷耐药或不稳定首选 |
| 不稳定单形VT(有脉) | 100J | YES | 能量递增（100→200→300→360J） |
| 多形VT/尖端扭转(有脉) | 200J | **NO（同步不可靠→按VF处理）** | MgSO₄ 2g IV 同时给药 |
