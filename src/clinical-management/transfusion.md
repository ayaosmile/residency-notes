# 输血指征与血液制品选择

## 结论

红细胞输注=**限制性策略（Hb<70→输血，目标70-90）**——心血管病患者<80→输血。FFP=仅活动性出血+凝血异常时（非经验性扩容）。PLT=治疗性<10-20或术前达标。遵循"输最少的血解决临床问题"——每一单位都有免疫调节风险。

## 核心要点

### 红细胞（RBC）

**限制性输血阈值 Hb<70 g/L（大多数患者）**
- **目标**：Hb 70-90 g/L（输1U RBC→升Hb≈10g/L）
- **急性冠脉综合征（ACS）**：Hb<80-90→输血（心肌需要氧——血氧输送不能太低）
- **血流动力学稳定+Hb 70-80**：不输（等待身体自身造血+补铁）

**"1次1单位"原则**
- 先输1U→复查Hb→评估症状→再决定是否需要第2U（不自动输2U——减少不必要的暴露量50%）

### 血浆（FFP/FP24）

**FFP适应证=活动性出血+凝血障碍——不是容量扩张剂！**

- **活动性出血+PT/APTT>1.5×正常** → FFP 10-15 mL/kg
- **大量输血（MTP）**：RBC:FFP:PLT=1:1:1——非等待凝血结果
- **华法林相关出血**：FFP已基本被PCC取代（PCC速效+病毒安全+用少液量）。FFP仅限PCC不可及时替代
- **非出血患者单纯INR↑**：不输FFP——INR轻度升高不是出血的充分条件

### 血小板（PLT）

| 临床场景 | 输注阈值（PLT×10⁹/L） |
|---------|---------------------|
| **无出血+稳定**（预防性） | **<10** |
| **发热/感染/小操作（非神经）** | <20 |
| **大手术/腰椎穿刺/中心静脉置管** | **<50** |
| **神经外科/眼科/颅内出血** | **<100** |
| **活动性出血** | <50（大出血目标>100） |
| **抗血小板药物+颅内出血** | 输1U血小板（即使PLT正常——部分逆转阿司匹林效应） |

- 1U机采血小板→升PLT≈20-30×10⁹/L
- 免疫性血小板减少（ITP/HIT/TTP）→**不输PLT（输血无效+可能加重血栓/溶血）**——病因治疗

### 冷沉淀/纤维蛋白原浓缩物

- **FIB<1.0-1.5 g/L+活动性出血**→冷沉淀1U/5-10kg（成人通常8-10U）或纤维蛋白原浓缩物4-6g IV
- **产科大出血/创伤DIC相关低纤维蛋白原**→尤其重要（纤维蛋白原消耗极快→优先补充纤维蛋白原而不是先补FFP）

### 输血并发症速查

| 并发症 | 时间 | 表现 | 处理 |
|--------|------|------|------|
| **急性溶血反应（ABO血型不合）** | **数分钟内** | 发热/寒战+胸痛+低血压+血红蛋白尿（尿液深色/酱油色）+DIC | **立即停止输血！** 维持血压+大量输液+利尿保护肾。送血库重做交叉配血 |
| **过敏反应** | 数分钟-1h | 皮疹→支气管痉挛→过敏性休克 | 轻症→抗组胺。重症→肾上腺素+中止输血 |
| **TRALI（输血相关急性肺损伤）** | **<6h** | 急性低氧+双肺浸润+无左房高压 | 支持（给氧±机械通气）。大多数48-96h恢复 |
| **TACO（输血相关循环超负荷）** | **<12h** | 呼吸窘迫+高血压+BNP↑ | **利尿剂+减慢/停止输血**。预防：高危患者输血前利尿+1mL/kg/h慢输 |
| **迟发性溶血反应** | 3-14d | 发热+贫血+Hb下降+黄疸 | 支持+避免再次输相同抗原RBC |

### 血液制品管理原则

- **知情同意**：除急诊外（患者无法签字→紧急输血授权审批流程），输血前必须签署知情同意
- **输血前核对**：2名医护人员床旁核对患者身份+血袋号+血型+有效期——**双人核对不可省略**（最核心的安全关卡）
- **输血速率**：血流动力学稳定→1U/2-4h。大出血→快速输注（加压袋+加温器）
- **输血开始后15min内密切观察**（大多数严重反应在此窗口）
