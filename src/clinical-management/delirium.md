# 谵妄的识别与处理

## 结论

谵妄 = **急性起病+波动性注意力障碍+认知改变（定向/记忆/语言/知觉）**→ 不可用既往认知障碍解释。分三型：过度活跃型（躁动）≈25%，**低活跃型（安静/"乖"）≈50%最常漏诊！**，混合型≈25%。常规用CAM/CAM-ICU筛查每一个住院患者。

## 核心要点

### 诊断（CAM-ICU）

满足 **特征1+2+（3或4）**：

1. **急性起病+波动性病程**
2. **注意力缺损**（不能完成连续字母测试或图片辨认）
3. **思维紊乱**（对话不连贯、跑题）
4. **意识水平改变**（RASS≠0=嗜睡或躁动）

> **低活跃型谵妄**：安静、反应慢、嗜睡→极易被当成"病人很配合/累了"而漏诊。RASS -1至-3 + CAM阳性 = **低活跃型谵妄——死亡率不亚于躁动型！**

### 鉴别诊断

**谵妄 vs 痴呆**

| | 谵妄 | 痴呆 |
|---|---|---|
| **起病** | 急性（数小时-数天） | 慢性渐进（数月-年） |
| **病程** | **波动性（经典型=日落加重/夜间躁动）** | 稳定渐进 |
| **注意力** | **明显受损** | 早期可正常 |
| **意识** | 改变（嗜睡至躁动） | 早期清楚 |
| **可逆性** | **可完全恢复**（识别+纠因） | 不可逆 |

**排查谵妄原因(DELIRIUM记忆法)**
- **D**rugs（药物——苯二氮䓬/抗胆碱/AChEI停药/阿片/激素/抗生素/多药并用）
- **E**lectrolyte（电解质：低钠/高钙）+ **E**ndocrine（内分泌：甲亢/甲减/肾上腺）
- **L**ow perfusion（低灌注：休克/心衰）+ **L**iver（肝性脑病）
- **I**nfection（感染——UTI/肺炎是住院谵妄第一大原因）+ **I**ntracranial（颅内：卒中/SAH/占位）
- **R**etention（尿潴留/粪便嵌塞）+ **R**espiratory（低氧/CO₂潴留）
- **U**ndiagnosed pain（未控制的疼痛——ICU谵妄最重要原因之一！）
- **M**etabolic（代酸/高血糖/低血糖/维生素B1/B12缺乏）

### 处理：非药物优先

**预防（循证6大措施）**
1. 规律昼夜节律——白天光照/开窗、夜间暗灯——**最有效最简单**
2. 早期活动——POD1坐椅+tid步行
3. 感觉辅助——患者自己的眼镜/助听器（别"收起来"！）
4. 规律作息——固定进食/活动/睡眠时间
5. 认知刺激——和患者对话/家人探视
6. 减少物理约束——**约束带增加谵妄和损伤风险**

**已发生谵妄的处理**
- **纠因第一**：上述DELIRIUM排查→感染（查UTI/CXR）、低氧、尿潴留（超声/一次性导尿）、便秘（指诊+通便）、未控制疼痛（镇痛！）
- **环境干预**：单人房间（减少过度刺激）、光照调整、专人陪护（家属>护工）

**药物——仅限严重躁动（威胁自身/他人安全时）**
- **氟哌啶醇 0.5-1mg PO/IV q4-12h**（最大<5mg/d）——一线（监测QTc，延长>500ms→停药）
- **右美托咪定 0.2-0.7 μg/kg/h IV**（ICU带管谵妄——唯一有循证证据的谵妄治疗药物）
- **奥氮平 2.5-5mg PO qhs**（替代氟哌啶醇）
- **避免苯二氮䓬！** 地西泮/咪达唑仑/**劳拉西泮增加谵妄风险和严重度**（仅限酒精戒断和BZD戒断谵妄时使用）

### 酒精戒断性谵妄（DT）——例外！

- **固定剂量苯二氮䓬是标准治疗**（劳拉西泮/地西泮/咪达唑仑，症状触发给药）
- CIWA-Ar评分（Clinical Institute Withdrawal Assessment）指导：≥8→启动BZD治疗，q1-2h评估
- 同时补充维生素B1 500mg IV q8h（预防Wernicke脑病+Korsakoff综合征）
