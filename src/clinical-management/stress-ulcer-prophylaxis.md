# 应激性溃疡的预防

## 结论

应激性溃疡预防（SUP）= **仅用于有明确指征的高危患者**（机械通气>48h/凝血障碍/休克/大烧伤等）→ 一线=**PPI**（奥美拉唑/泮托拉唑）或 **H2RA**（法莫替丁）。非高危患者不常规SUP——增加肺炎+艰难梭菌感染风险。

## 核心要点

### 哪些患者需要SUP？

**主要高危指征（满足1条即启动）**

| 高危因素 | 说明 |
|---------|------|
| **机械通气>48h** | 最强的独立危险因素（+凝血障碍=极高危） |
| **凝血障碍** | INR>1.5 或 PLT<50×10⁹/L 或 APTT>2×正常值 |
| **休克** | 持续低血压/需升压药 |
| **严重烧伤**（TBSA>35%） | Curling溃疡高危 |
| **严重创伤**（ISS≥16/颅脑损伤GCS≤10） | Cushing溃疡高危 |
| **急性肝衰竭/肝硬化急性失代偿** | 门脉高压性胃病出血高危 |
| **AKI需RRT** | 尿毒症性胃炎+持续抗凝 |

**多项高危因素叠加→SUP必需启动**；仅1-2条低危→不常规SUP。

**不需SUP的患者**
- 普通病房可进食的术后患者（大多不需要）
- 仅用抗凝/抗血小板单药（无其他高危因素）

### 药物选择

| 药物类别 | 代表药 | 剂量 | 特点 |
|---------|--------|------|------|
| **PPI**（一线） | 泮托拉唑 40mg IV/PO qd 或 奥美拉唑 20-40mg IV/PO qd-bid | — | 抑酸强效。**快速降酸**→ICU首选。可能增加肺炎+艰难梭菌风险 |
| **H2RA** | 法莫替丁 20mg IV/PO q12h | — | 抑酸中度。PEPTIC试验(2020):H2RA与PPI死亡率一致但上消化道出血略多于PPI |
| **硫糖铝** | 1g PO q6h（餐前1h+睡前） | — | 覆盖黏膜屏障，不升高胃pH→肺炎/艰难梭菌风险较低。出血预防证据弱于PPI/H2RA |

### 药物过渡与停药

- **气管拔管/停升压药/凝血障碍恢复/开始肠内营养** → 停SUP（大多数患者此时风险不再高危）
- ICU转出时：评估是否仍需SUP——不自动带出院
- **PPI短期使用（<7d）→骤停安全**。长期（>4周）→渐减量（否则酸反跳）→H2RA过渡

### 关键风险

- **肺炎**：胃酸被抑制→胃内细菌定植↑→误吸后肺炎风险↑（任何抑酸药均增加）
- **艰难梭菌感染**：胃酸缺失→艰难梭菌孢子存活+繁殖→假膜性肠炎。**尽量减少PPI非必要使用——就是最重要的预防**
- **PPI长期副作用**：B12缺乏、低镁、骨质疏松（髋部骨折风险↑）、CKD进展
