# 营养支持原则

## 结论

营养支持 = **肠内优先（"If the gut works, use it"）→ 肠外仅限肠内禁忌/不耐受**。非禁食患者：**24-48h内启动肠内营养**（重症也一样——不等肠鸣音！）。肠内禁忌（肠梗阻/穿孔/严重休克/肠缺血）→**7天内启动肠外**。

## 核心要点

### 营养途径选择决策树

```
能经口吗？
├─ 能→ 经口饮食（首选）
└─ 不能→ 肠道有功能吗？
    ├─ 有→ 肠内营养（鼻胃/鼻空肠管）
    └─ 无（梗阻/穿孔/高流量瘘/肠缺血/严重休克）→ 肠外营养（PN）
```

### 能量与蛋白目标

| 参数 | 目标 | 如何计算 |
|------|------|---------|
| **总能量** | 25-30 kcal/kg/d | 理想体重或调整体重（肥胖BMI>30→22-25kcal/kg理想体重） |
| **蛋白质** | **1.2-2.0 g/kg/d**（重症） | 非重症：0.8-1.0 g/kg/d；外科/创伤：1.5-2.0 g/kg/d；烧伤：2.0-2.5 g/kg/d |
| **碳水化合物** | 占总热卡 50-60% | — |
| **脂肪** | 占总热卡 20-30% | — |
| **水** | 25-30 mL/kg/d | 另计异常丢失+发热额外补充 |

### 肠内营养（EN）启动与进展

**启动（<24-48h）**
- **不需要等肠鸣音恢复！**（小肠蠕动在术后数小时即恢复，结肠需要48-72h。听肠鸣音来"决定能不能喂"已过时——证据：早期EN改善预后）
- 鼻胃管（NG/OG）vs鼻空肠管（NJ）：NG优先（放置简单+非胰腺炎患者效果同NJ）
- 起始速率：**10-20 mL/h**→耐受→q4-8h增加10-20mL/h至目标速率

**监测**
- **胃残余量（GRV）q4-6h**——GRV<250-500mL→继续；GRV>500mL→促动力药（甲氧氯普胺10mg IV q6-8h或红霉素3mg/kg IV q6h）。不因单次高GRV直接停EN——先用促动力药
- 腹胀/呕吐/腹泻→下调速率→排查原因→再次尝试

**EN禁忌——转PN**
- 机械性肠梗阻（完全）
- 肠穿孔/吻合口漏（未控制）
- 严重休克（血流动力学不稳定+大剂量升压药→EN易诱发肠缺血）
- 肠缺血/坏死
- 高流量肠瘘（无法在瘘远端置管）
- 严重腹胀+腹腔间隔室综合征(ACS)

### 肠外营养（PN）

**适应证**：EN禁忌预计>7天；围术期严重营养不良+EN禁忌；短肠综合征。

- **外周PN**（PICC/外周静脉）：渗透压<900mOsm/L，≤14d使用。适合短期、低渗透压（限脂量）
- **中心静脉PN**：渗透压>900mOsm/L，>14d使用（放置PICC/CVC）

**PN要素**：

| 组分 | 示例 |
|------|------|
| 氨基酸 | 10% 500mL=50g蛋白 |
| 葡萄糖 | 50%GS 500mL=250g葡萄糖=1000kcal（不超过5mg/kg/min=避免脂肪肝+高血糖） |
| 脂肪乳 | 20%脂肪乳 250mL=500kcal（提供必需脂肪酸+节省蛋白质。占总热卡≤30%） |
| 电解质+微量元素+维生素 | 钠、钾、钙、镁、磷、锌、硒、VitB1(硫胺素！——葡萄糖代谢必需)、VitK |

> **再喂养综合征(Refeeding Syndrome)！** 重度营养不良+NPO>5d→启动营养72h内→**低磷+低钾+低镁+维生素B1缺乏→心衰+呼衰+心律失常**。预防：前3天低热卡启动（10-15kcal/kg→逐渐增加）+营养前纠正电解质+补充维生素B1 200-300mg/d IV。

### 特殊营养需求

- **ARDS/重症**：免疫调节EN（含ω-3鱼油+γ-亚麻酸+抗氧化剂）→改善氧合、缩短机械通气时间
- **烧伤**：极高蛋白2.0-2.5g/kg/d+高热量
- **肝衰**：支链氨基酸（BCAA）配方——减少肝性脑病发作
- **AKI/CKD**：低钾+低磷+低镁配方。RRT→高蛋白丢失（透析液中++）→蛋白需求升至2.0-2.5g/kg/d
