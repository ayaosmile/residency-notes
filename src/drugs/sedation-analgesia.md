# 镇静镇痛药物速查

## 结论

镇痛优先于镇静（**Analgosedation原则**）——先给镇痛药，镇静药辅助。ICU 镇静目标：RASS -2 到 0（轻镇静→清醒合作）。常规避免持续深度镇静（增加谵妄+死亡率）。

## 镇痛药

| 药物 | 常用剂量 | 起效/时效 | 特点 |
|------|---------|---------|------|
| **吗啡** | 2-5 mg IV q5-15min 滴至止痛；持续 2-10 mg/h | 5-10min / 4-5h | 组胺释放→低血压。肾功能不全→活性代谢物蓄积 |
| **芬太尼** | 25-50 μg IV q5-15min；持续 50-200 μg/h | 1-2min / 30-60min | 血流动力学影响小（首选心源性/休克患者）。半衰期随输注时间延长 |
| **吗啡酮（氢吗啡酮）** | 0.5-1 mg IV q10min | 5-10min / 4-5h | 比吗啡少组胺释放。无活性代谢物 |
| **瑞芬太尼** | 0.05-0.3 μg/kg/min（无bolus） | 1-3min / 3-5min | 被血/组织酯酶快速代谢（不依赖肝肾！）→ 停药后几分钟内苏醒 |
| **舒芬太尼** | 0.5-5 μg IV bolus；持续 10-50 μg/h | 1-2min / 1-2h | 比芬太尼强5-10倍 |

## 镇静药

| 药物 | 常用剂量 | 起效 | 特点 |
|------|---------|------|------|
| **丙泊酚** | 0.5-4 mg/kg/h 持续输注 | 15-30s | **首选短期ICU镇静**——快速起效/快速苏醒。心功能抑制+高甘油三脂→超过48h警惕**丙泊酚输注综合征**（PRIS） |
| **右美托咪定** | 0.2-1.4 μg/kg/h（无bolus/减量bolus） | 15-30min | α2激动剂——镇静+镇痛+不抑制呼吸！**减少谵妄**。注意心动过缓 |
| **咪达唑仑** | 1-5 mg IV q10min；持续 1-5 mg/h | 1-3min / 30-60min | 苯二氮䓬——**减少谵妄不推荐长期**。CKD→活性代谢物蓄积 |
| **劳拉西泮** | 1-4 mg IV q6-8h（非持续） | 15-20min / 4-6h | 长效。ICU谵妄首选BZD |

## 常见临床场景快速选择

| 场景 | 推荐方案 |
|------|---------|
| **ICU短期镇静（<48h）** | 丙泊酚 0.5-4 mg/kg/h + 芬太尼 50-200 μg/h |
| **整体ICU镇静（>48h）** | 右美托咪定 0.2-1.4 μg/kg/h + 芬太尼/氢吗啡酮（减少谵妄+通气时间） |
| **心源性/低血压患者** | 芬太尼（优先，血流动力学稳定） + 必要时小剂量咪达唑仑 |
| **操作镇静（内镜/复位）** | 丙泊酚 0.5-1 mg/kg IV bolus（分次推注） + 芬太尼 0.5-1 μg/kg |
| **机械通气不耐受（烦躁）** | 镇痛先→芬太尼 25-50 μg IV + 丙泊酚 10-20 mg IV push → 持续输注 |
| **酒精戒断** | 劳拉西泮/地西泮（BZD，症状触发给药优于固定方案） |

## 关键原则

- **镇痛优先（Analgosedation）**：所有ICU患者都应先评估疼痛 → 处理疼痛 → 再镇静
- **每日唤醒（SAT）**：每天停或减镇静药 → 评估神经状态 → 自主呼吸试验（SBT）→ 减少通气时间和ICU住院日
- **老年/衰弱患者**：剂量半量起步（右美托咪定 0.2-0.7 μg/kg/h；丙泊酚 0.3-2 mg/kg/h；阿片减半）
- **谵妄预防**：右美托咪定 > 丙泊酚 > 苯二氮䓬（BZD使用增加谵妄风险）。规律昼夜光照、早期活动、家人参与
