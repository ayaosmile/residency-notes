# 经验性抗生素选择速查

## 结论

经验性抗生素 = **感染部位 + 社区/院内 + 严重程度 + 本地区耐药监测** 决定方案 → 使用前取血培养（2套）→ 48-72h 后按培养结果降阶梯。

## 常见感染经验性方案

| 感染部位 | 社区获得性（轻-中症） | 重症/耐药风险/院内 |
|---------|-------------------|-----------------|
| **社区获得性肺炎（CAP）** | 阿莫西林克拉维酸 1.2g q8h IV 或 头孢曲松 2g q24h IV + 阿奇霉素 500mg q24h | 哌拉西林他唑巴坦 4.5g q6h 或 头孢吡肟 2g q8h + 阿奇霉素/莫西沙星 |
| **泌尿系感染** | 头孢呋辛 0.25g bid PO（膀胱炎）; 头孢曲松 2g q24h IV（肾盂肾炎） | 哌拉西林他唑巴坦 或 碳青霉烯（ESBL风险） |
| **腹腔感染** | 头孢曲松 2g q24h + 甲硝唑 500mg q8h | 哌拉西林他唑巴坦 4.5g q6h 或 厄他培南 1g q24h |
| **皮肤软组织感染** | 头孢唑林 2g q8h IV（蜂窝织炎）; 双氯西林 0.5g qid PO（轻度） | 万古霉素 15-20mg/kg q8-12h（MRSA风险） + 哌拉西林他唑巴坦（需覆盖厌氧菌+革兰阴性） |
| **粒缺伴发热** | — | **立即**头孢吡肟 2g q8h 或 哌拉西林他唑巴坦 4.5g q6h 或 美罗培南 1g q8h ± 万古霉素（导管/皮肤感染风险） |
| **脑膜炎** | 头孢曲松 2g q12h + 万古霉素 15-20mg/kg q8-12h | + 氨苄西林 2g q4h（>50岁/免疫抑制→覆盖李斯特菌） |

## 关键原则

- **CAP的一口二口之别**：之前用过抗生素/住院 > 48h = 耐药风险 → 升级至抗假单胞菌方案
- **ESBL风险**：近3个月内使用过3代头孢/喹诺酮、长期住院/护理机构、已知ESBL定植 → 碳青霉烯类
- **MRSA风险**：已知MRSA定植/感染史、导管感染、化脓性皮肤感染、血液透析 → 加用万古霉素/利奈唑胺/达托霉素
- **真菌风险**：粒缺 > 7天、长期激素/免疫抑制、广谱抗生素使用 > 1周无反应 → 加用棘白菌素（卡泊芬净 70mg→50mg q24h）
- **别忘停药原则**：培养阴性+PCT < 0.25/降≥80%+临床改善 → 48-72h后停用或降阶梯（限制广谱抗生素使用 ≤ 7天）
