# 血管活性药物速查

## 结论

血管活性药根据**受体效应选择**——去甲肾（α1+β1）= 感染性休克一线；肾上腺素（α1+β1+β2）= 过敏性休克+心脏骤停首选；多巴酚丁胺 = 心源性休克强心；多巴胺 = 已不再一线。

## 核心药物速查表

| 药物 | 常用剂量 | 受体效应 | 主要用途 |
|------|---------|---------|---------|
| **去甲肾上腺素** | 0.05-1.0 μg/kg/min | α1 ≫ β1 | **脓毒性休克一线**——强力缩血管。经中心静脉给药；短期可经粗大外周静脉 |
| **肾上腺素** | 0.05-2.0 μg/kg/min（输注）；0.3-0.5mg IM（过敏） | α1 = β1 = β2（全激动） | 过敏性休克（IM首选）+ 心脏骤停 + 心源性休克难治性 |
| **多巴酚丁胺** | 2.5-20 μg/kg/min | β1 ≫ β2 ≫ α | **心源性休克**——正性肌力+轻度血管扩张+心率↑（不超过SBP的代偿性↓） |
| **血管加压素** | 0.01-0.04 U/min（固定） | V1 ≫ V2 | 难治性脓毒性休克的去甲肾辅助——降低去甲肾剂量 |
| **多巴胺** | 2-20 μg/kg/min | D1→β1→α（剂量↑） | 已不一线。仅保留：症状性心动过缓+阿托品无效等待起搏时用 |
| **米力农** | 负荷 25-50 μg/kg 10min → 0.25-0.75 μg/kg/min | PDE3i（cAMP↑） | 慢性心衰急性失代偿+β阻滞剂基础——不依赖β受体 |
| **去氧肾上腺素** | 50-200 μg/min | α1纯 | 仅限：严重主动脉瓣狭窄+低血压、或去甲肾禁忌（心动过速难以耐受） |

## 受体效应速记

| 受体 | 效应 |
|------|------|
| α1 | 血管收缩 → SBP↑ + DBP↑（后负荷↑） |
| β1 | 心脏 → 心率↑ + 心肌收缩力↑（正性肌力） |
| β2 | 血管舒张 → SVR↓（后负荷↓）+ 支气管舒张 |
| D1 | 内脏/肾血管舒张（多巴胺低剂量——非"肾保护"！） |
| V1 | 血管平滑肌收缩 → SVR↑ |

## 关键原则

- **中心静脉 vs 外周静脉**：升压药倾向经中心静脉给药。但紧急时（等不到中心静脉）→ 近端粗大外周静脉（肘前/颈外）+ 短期（< 24h）+ 严密监测外渗
- **MAP 目标**：一般达 65 mmHg。慢性高血压 → 可更高（但不应单独为了数字过度升压）
- **感染性休克 1h Bundle**：液体 30 mL/kg → 仍低血压 → 启动去甲肾（不要等液体复苏"完成"）→ MAP<65 → 加血管加压素（去甲肾 > 0.25-0.3 μg/kg/min 后）
- **心源性休克**：多巴酚丁胺一线。伴心衰+低血压 → 去甲肾+多巴酚丁胺联用。不推荐多巴胺
- **撤药顺序**：血流动力学稳定 > 24h → 先撤血管加压素 → 再逐渐减停去甲肾（不要同时停）
