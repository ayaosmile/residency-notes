# 胰岛素方案速查

## 结论

住院患者高血糖 → **胰岛素首选**（口服药暂停或减量）。重症/进食不稳定 → 持续 IV 输注；非重症 → 基础+餐时方案（Basal-bolus）。**低血糖是最大风险——每 4-6h 测血糖！**

## 胰岛素类型速查

| 类型 | 代表药 | 起效 | 达峰 | 持续时间 | 用途 |
|------|--------|------|------|---------|------|
| **速效（Lispro/Aspart/Glulisine）** | 门冬胰岛素 | 5-15min | 1-2h | 4-5h | 三餐前餐时 + 校正量 |
| **短效（Regular）** | 普通人胰岛素 | 30min | 2-3h | 6-8h | 静脉输注（DKA/HHS首选）+ 三餐前（已普遍被速效替代） |
| **中效（NPH）** | 低精蛋白锌胰岛素 | 1-2h | 4-12h | 12-18h | 睡前基础（现在多被长效替代） |
| **长效（Glargine/Detemir）** | 甘精胰岛素 U100 | 1-2h | 几乎无峰 | ~24h | **每日一次基础**（最常用） |
| **超长效（Degludec/Glargine U300）** | 德谷胰岛素 | 1h | 无峰 | > 42h | 极平稳基础，低血糖更少。CKD友好 |

## 住院患者胰岛素方案

### 重症/禁食/静脉输注患者 — 持续 IV

**普通人胰岛素 0.1 U/kg/h IV 持续输注**（或 0.05-0.1 U/kg/h）
- 目标血糖：7.8-10.0 mmol/L。**每 1h 测血糖**（稳定后 q2h）
- 血糖 > 10.0 → 上调 0.5-1 U/h；血糖 7.8-10.0 → 维持；血糖 5.0-7.8 → 下调 0.5-1 U/h
- **血糖 < 5.0 或骤降 > 3 mmol/L/h → 减半或暂停 → 给糖**

### 非重症规律进食患者 — Basal-bolus

**总日剂量（TDD）= 0.4-0.5 U/kg/d**（老年人/肾病/脆弱者 0.2-0.3 U/kg/d）

| 组分 | 剂量 | 举例（70kg, TDD=28U） |
|------|------|---------------------|
| **基础（Basal）** | TDD × 50% = 甘精胰岛素睡前一次 | 14 U 睡前 |
| **三餐餐时（Bolus）** | TDD × 50% ÷ 3 = 速效胰岛素三餐前 | 每餐前 ~5 U |
| **校正量（Correction）** | 血糖 > 目标 → 速效胰岛素 1-2 U 附加。**校正因子（CF）= 1800 / TDD** ≈ 每 1U 降血糖 mmol/L | CF=1800/28≈64 → 1U≈降 2.8 mmol/L（需换算） |

### 过渡：IV → 皮下

- 在停止 IV 胰岛素之前 ≥ 2 小时给予首剂皮下基础胰岛素
- 按前 6h IV 胰岛素平均速率 × 24h × 80% = 皮下基础胰岛素首剂量

## 特殊场景

| 场景 | 方案 |
|------|------|
| **DKA/HHS** | 胰岛素 0.1 U/kg/h IV 持续输注。血糖降至 13.9 mmol/L → 换用 5%GS + 胰岛素 0.05 U/kg/h → 继续 IV 直到酮症消失/pH正常 |
| **围术期** | 术前当日基础胰岛素给 50-75% 原剂量。术中监测 q1h。术后过渡回原方案 |
| **激素使用中** | 地塞米松/甲泼尼龙 → 显著升高血糖（午后-傍晚高）→ 基础胰岛素可能需翻倍 + 加强餐时 |
| **禁食/肠外营养** | TPN 固定速率 → 在 TPN 袋中直接加胰岛素（初始 0.1 U/g 葡萄糖）或 长效胰岛素日一次 0.2-0.3 U/kg。每日调整 |
