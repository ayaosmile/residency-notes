# 术后出血的评估与处理

## 结论

术后出血 → **分原发（<24h）vs 继发（5-14d）→ 评估容量状态+出血速度** → 稳定+小量=保守输血+监测；不稳定/持续活动性出血 → **急诊再手术（不等待Hb降到某值才手术——出血的临床速度决定）**。

## 核心要点

### 原发 vs 继发出血

| 类型 | 时间 | 原因 | 处理倾向 |
|------|------|------|---------|
| **原发出血** | **<24h（通常<6h）** | 术中止血不彻底、结扎线滑脱、未识别的血管损伤 | 活动性出血→手术（探查止血） |
| **继发出血** | **POD 5-14** | 感染侵蚀血管（假性动脉瘤破裂）、吻合口出血、应激性溃疡 | 积极保守（感染灶中再次手术止血困难！）→介入栓塞首选+抗感染 |
| **凝血功能障碍性出血** | 任意时间 | 消耗性（DIC）、稀释性（大量输血后）、肝病基础、抗凝药 | **先纠凝血**（血小板+PCC+冷沉淀+Vit K/逆转剂）→ 凝血因子上来后仍出血→手术 |

### 出血严重度与处理

| 表现 | 处理 |
|------|------|
| **小量/中量+稳定** | **保守+输血+密切监护**。Hb<70→输血1-2U；PLT<50→输血小板。引流监测+连续Hct q4-6h。凝血异常→纠凝血。**临床稳定前提下可保守观察**。止血药物辅助（氨甲环酸1g IV q8h） |
| **持续活动性+不稳定** | **不等Hb低！启动大量输血方案+立即手术**。Hb每h下降>10g/L+引流>200mL/h×2-4h→手术无替代。**任何血流动力学不稳定=手术** |
| **迟发性出血（POD>7+脓肿/感染征象）** | **CTA→DSA介入栓塞**。感染环境再手术止血不可预测——假性动脉瘤破裂血管脆烂难以直接缝。同时抗生素控制感染 |

### 手术决策的"四问"

1. **血流动力学是否稳定？** 不稳定→**即刻手术**
2. **引流量速度？** >200mL/h超过2h→手术；<50mL/h+稳定→保守
3. **凝血功能？** PT↑/APTT↑/PLT↓→先纠凝血因子再评估是否仍需要手术
4. **术后时间？** <24h→手术积极；POD7-14+感染→介入优先

### 大量输血方案（MTP）

- **启动标准**：ABC评分≥2（穿透伤+FAST阳性+SBP≤90+HR≥120）——手术患者为持续活动性出血+低血压
- **成分比例**：**RBC : FFP : PLT = 1:1:1**
- **药物辅助**：氨甲环酸1g IV bolus（<3h内）→1g输注8h
