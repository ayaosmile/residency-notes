# 术后肠梗阻与肠麻痹的处理

## 结论

腹部术后**生理性肠麻痹**（小肠24h恢复→胃48h→结肠72h）→ 超过正常时间+腹膨隆+恶心呕吐= **术后肠梗阻（POI）** → 分麻痹性（90%）vs 机械性（10%=粘连/内疝）→ 保守治疗为主（NGT+限液+纠电解质+促动力+阿片减量）；CT排除机械性病因。

## 核心要点

### 正常胃肠道恢复时序

| 部位 | 恢复时间 | 恢复标志 |
|------|---------|---------|
| 小肠 | 数小时-24h | 可吸收（早期肠内营养的理论依据） |
| 胃 | 24-48h | NGT引流量减少、恶心减轻 |
| 结肠 | 48-72h | **排气=结直肠功能恢复**（不是"肠蠕动"） |

**排气后再进食**（不是"肠鸣音出现"——小肠恢复远早于结肠，只听肠鸣音等不科学）。

### 术后肠梗阻（POI）诊断

- **定义**：超越预期时间的腹胀、恶心呕吐、不排气、不能耐受经口饮食
- **CT（腹部+盆腔）**：区分麻痹性（弥漫性肠扩张无移行区）vs 机械性（移行区/闭环/内疝）→ 后者需手术

### POI处理

| 措施 | 细节 |
|------|------|
| **NGT减压** | 腹胀明显+恶心呕吐→NGT持续低负压。引流量减少→可夹管观察→可拔管 |
| **液体管理** | 限液（euvolemia非超负荷）+纠电解质（低钾、低镁、低磷加重肠麻痹→必须在正常范围） |
| **阿片类减量** | **POI最核心的预防+治疗=阿片最小化**。NSAIDs/对乙酰氨基酚/加巴喷丁替代阿片。硬膜外镇痛（局麻药↛阿片）→POI↓ |
| **促动力药物** | 爱维莫潘(alvimopan)12mg PO preop→postop bid×7d（选择性μ受体拮抗剂——肠道促动力不进入CNS）。甲氧氯普胺10mg IV q6-8h（用于胃瘫） |
| **咀嚼口香糖** | 假饲效应——咀嚼15min tid→早排气+住院缩短（meta分析证实）——零成本零副作用 |
| **早期活动** | 下床活动tid→促进胃肠蠕动（非独立证据但属于ERAS核心措施） |
| **进食** | 一旦排气→清流或软食（无需"流质→半流→软食"递进——可跳级） |

### POI vs 机械性梗阻的鉴别

| | 麻痹性POI | 机械性梗阻 |
|---|---|---|
| **起病** | 术后持续不缓解（原发） | 突然加重或先排便后再次梗阻 |
| **腹痛** | 弥漫腹胀+不适 | 绞痛+阵发性加重（肠痉挛试图推进） |
| **CT** | 弥漫性肠扩张无移行区 | 明确移行区/闭环/内疝/粘连束带 |
| **处理** | 保守 | 手术/再次剖腹 |
