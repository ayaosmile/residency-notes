# 吻合口漏的识别与处理

## 结论

胃肠道术后 **POD5-7突发腹痛+发热+心动过速±引流液呈胆汁/粪样** → 吻合口漏 → **急诊CT（水溶性造影剂灌肠/口服，非钡剂！）→ 分型处理**：小漏+稳定=经皮引流+抗生素+禁食TPN；大漏+不稳定/腹膜炎→急诊手术。

## 核心要点

### 诊断

- **典型时间窗**（吻合口漏"5-7天规则"）：胶原溶解期（POD 3-7吻合口强度最低——因胶原酶活性高峰）→结直肠术后POD5-7、胃/小肠术后POD3-5漏
- **临床预警信号**：
  - 早期（不易识别）：不明原因的心动过速、呼吸急促、轻度腹胀——**在血流动力学恶化之前先出现心率/呼吸变化（最早24-48h的信号！）**
  - 典型：突发腹痛+发热+WBC↑↑+引流液突然增多且浑浊（胆汁/粪样=肠道漏；浑浊乳糜样=淋巴漏/胰漏）
  - 晚期：脓毒性休克（延迟诊断）
- **CT（口服水溶性造影剂）**：造影剂外溢至腹腔=确诊。**禁止钡剂灌肠**（钡剂漏入腹腔→重度化学性腹膜炎）
- **术后腹腔引流液淀粉酶/肌酐/尿素测定**：胰漏→引流淀粉酶>血清3倍；尿漏→引流肌酐/尿素>血清水平

### 处理决策

| 类型 | 处理 |
|------|------|
| **小漏+稳定+已引流（Contained leak）** | **保守**（70%可保守成功）：禁食+TPN+广谱抗生素（哌拉西林他唑巴坦/美罗培南）×7-10d + 经皮引流（CT/超声引导）→ 引流管保持至漏口闭合（出院带管管理） |
| **大漏+不稳定/弥漫性腹膜炎（Free leak）** | **急诊手术** → ①切除吻合口±重做吻合（组织健康+稳定+污染轻）→+保护性造口 ②或 Hartmann式末端造口（污染重/不稳定） |
| **吻合口漏+脓毒性休克** | **Damage control surgery**——首先控制污染（腹腔冲洗引流+末端造口）→ ICU继续复苏 → 24-48h后二次探查 |

### 吻合口漏风险因素

- **术前**：低蛋白（Alb<30g/L）、贫血、糖尿病、吸烟、肥胖（BMI>30）、新辅助放疗（组织微循环！）
- **术中**：吻合口张力、血供不良（结直肠吻合漏率：高位>5-8%、低位>10-15%）、吻合器操作失误、污染
- **术中预防**：吻合后测漏试验（充气/注射染料）、**低位结直肠吻合常规保护性造口**（距肛缘<6cm的吻合口必做）、网膜包裹吻合口

### 关键数字

- **结直肠术后漏率**：择期 3-8%（直肠癌低位前切除漏率可达 10-20%）
- **一旦发生漏 → 30天死亡率 10-20%**（如果发展为脓毒性休克可达30-40%）
- **延迟诊断（>24h）→ 死亡率翻倍**
- **保护性造口的作用**：降低漏的严重度（不降低发生率）→有了造口即使发生漏也常可保守处理而不需二次大手术
