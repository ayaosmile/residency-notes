# 术后DVT/PE的预防与识别

## 结论

术后**Caprini评分≥3=中高危 → 必须预防**（机械+药物LMWH）→ POD5-14胸痛+低氧+单侧腿肿 → Wells评分 → **D-二聚体+超声/CTPA排查** → 确诊DVT/PE=立即抗凝（LMWH/DOAC）。

## 核心要点

### Caprini评分与预防分层

| Caprini总分 | 风险等级 | VTE发生率（无预防）| 预防方案 |
|------------|---------|------------------|---------|
| 0 | 极低 | <0.5% | 早期活动（不需药物） |
| 1-2 | 低 | ~1.5% | 机械预防（IPC弹力袜） |
| **3-4** | **中** | ~3% | **LMWH/UFH**（术后6-12h启动）+ 机械预防 |
| **≥5** | **高** | ~6% | **LMWH bid** 或 UFH tid + IPC。延长预防至28d（腹部/盆腔癌症手术） |

> 大多数腹部/盆腔开放手术 Caprini ≥5（手术时间>45min+年龄>60+恶性肿瘤=轻易≥5分）→ **常规药物预防**

### DVT诊断

- **Wells DVT评分**：活动性癌(1)+麻痹/制动(1)+卧床>3d/大手术4周内(1)+沿深静脉压痛(1)+全腿肿胀(1)+小腿肿胀>对侧3cm(1)+凹陷性水肿(1)+浅静脉侧支(1)+既往DVT(1)+DVT与别诊断同等可能(-2)
  - **≥2分=DVT可能** → 超声
  - **<2分=DVT不太可能** → D-二聚体。阴性+D-二聚体<500→排除DVT
- **超声**（近端静脉加压超声=金标准）——股静脉+腘静脉不可压缩=DVT

### PE诊断

- **Wells PE评分**：DVT临床征象(3)+PE与别诊断同等可能(3)+HR>100(1.5)+制动/手术4w内(1.5)+既往DVT/PE(1.5)+咯血(1)+活动性癌(1)
  - **>4分=PE可能** → CTPA
  - **≤4分=PE不太可能** → D-二聚体 → 年龄校正阴性值排除PE
- **CTPA**（金标准）→肺动脉内充盈缺损=PE
- **床旁评分+超声+CTPA**：PESI/sPESI简化PE严重度评分——**sPESI≥1→高危/中高危→收住院**

### DVT/PE治疗

| 临床状态 | 方案 |
|---------|------|
| **PE+低血压/休克** | **溶栓（阿替普酶100mg IV 2h）+ 肝素/DOAC** |
| **PE稳定** | 依诺肝素1mg/kg q12h 或 利伐沙班15mg bid×21d→20mg qd |
| **近端DVT** | 依诺肝素1mg/kg q12h 或 利伐沙班15mg bid×21d→20mg qd → 至少3-6个月抗凝 |
| **远端DVT（小腿孤立）** | ①抗凝 3个月 或 ②复查超声 q1-2周（不抗凝但密切观察——如近端进展→转为抗凝） |
| **术后患者抗凝特殊情况** | 颅内/脊柱手术后→ IVC滤器（临时、可回收）替代抗凝（抗凝禁忌期） |

### 药物预防用法

| 药物 | 剂量 | 频次 | 启动时间 |
|------|------|------|---------|
| 依诺肝素 | 40mg SC | qd（中危） | 术后6-12h（椎管内麻醉12h后） |
| 依诺肝素 | 30mg SC | q12h（高危/肥胖BMI>40）| 同上 |
| 普通肝素 | 5000U SC | q8h | 同上 |
| 磺达肝癸钠 | 2.5mg SC | qd | 术后6-8h |
