# 术后肺部并发症的预防与处理

## 结论

术后肺部并发症=**肺不张（最常见POD1-2）→肺炎→呼衰→ARDS（最严重）逐级恶化**。预防决定结局——**激励性肺活量计（IS）+早期下床+疼痛控制** 三要素降低>50%并发症。一旦进展到肺炎/呼衰→抗生素+氧疗±无创通气。

## 核心要点

### 常见肺部并发症

| 并发症 | 时间 | 诊断 | 处理 |
|--------|------|------|------|
| **肺不张** | POD 1-2（最常见） | 低热（<38.3°C）+ 干咳 + CXR基底不张/膈肌抬高 | **IS q1h清醒时 + 深呼吸/咳嗽 + 早期下床**。不需抗生素、不需支气管镜 |
| **肺炎** | POD 3-7 | 发热>38.5°C + 脓痰 + CXR新浸润 + WBC↑/核左移 | 抗生素（见下）+ 雾化排痰（NS 5mL+AAC雾化q8h）+ IS+物理治疗 |
| **呼衰（需给氧/通气）** | POD 2-7 | PaO₂<60 / SaO₂<90%尽管给氧 / PaCO₂>50+pH<7.30 | 面罩/Venturi→HFNC→NIV。NIV失败→插管 |
| **ARDS** | POD 3-10（脓毒症/大量输血/误吸后） | P/F<300+双肺浸润+无左房高压 | 保护性通气（Vt 6mL/kg IBW+Pplat<30）+ 俯卧位（如果P/F<150） |

### 肺不张 vs 肺炎鉴别

| | 肺不张 | 肺炎 |
|---|---|---|
| **时间** | POD 1-2（最早） | POD 3-7 |
| **体温** | <38.3°C（低热） | >38.5°C（高热+寒战） |
| **痰** | 干咳或少量黏液 | 脓痰（黄/绿色） |
| **WBC** | 正常或微升 | 核左移+WBC>12×10⁹/L |
| **CXR** | 基底条状不张+膈肌抬高+肋膈角钝 | 局灶浸润/实变+空气支气管征 |
| **治疗** | IS+深呼吸+活动（不需抗生素） | 抗生素+雾化+IS+物理治疗 |

### 抗生素方案（术后肺炎）

| 类型 | 方案 |
|------|------|
| **社区获得性（<5d住院/健康/无近期抗生素）** | 头孢曲松2g q24h+阿奇霉素500mg q24h IV |
| **院内/耐药风险（>5d住院/近期抗生素/免疫抑制/COPD基础）** | 哌拉西林他唑巴坦4.5g q6h 或 头孢吡肟2g q8h ± 万古霉素（MRSA风险） |
| **误吸肺炎（胃内容物吸入）** | 哌拉西林他唑巴坦 或 头孢曲松+甲硝唑。不需覆盖厌氧菌（证据不支持） |
| **痰培养指导降阶梯** | 48-72h痰培养回报→降窄谱 |

### 预防（最有效的干预）

| 措施 | 细节 |
|------|------|
| **IS**（激励性肺活量计）| **q1h清醒时**×10次——最重要。必须在术前教会患者！ |
| **早期下床** | POD1即下床坐椅+tid步行（ERAS强制要求） |
| **深呼吸+咳嗽** | 有效咳嗽前提=无痛——**胸腹手术患者必须充分镇痛**（硬膜外/NSAIDs） |
| **术后FiO₂** | **避免不必要的术后高浓度氧**（SpO₂ 94-98%为目标——自由氧→吸收性肺不张）。术后以最低FiO₂维持SpO₂目标 |
| **术前戒烟** | 术前戒烟4-6周降>50%肺部并发症。戒烟时间越长越好 |
| **口腔卫生** | 术前洗必泰漱口（心脏和胸科手术证据最强）+术后每日氯己定漱口 |
| **误吸预防** | 床头抬高30°、胃管进食前回抽确认位置、拔管前空腹 |
