# 贡献指南

感谢你对规培笔记的关注。

## 参与方式

### 报告错误或提出建议

在 [Issues](https://github.com/your-username/residency-notes/issues) 中提交，请包含：
- 具体页面链接
- 问题描述或改进建议
- 如有，附上参考文献/来源

### 提交新内容或修改

1. Fork 本仓库
2. 创建你的分支 (`git checkout -b fix/your-fix`)
3. 遵循知识卡片格式规范（见下方）
4. 提交 PR 到 `main` 分支

## 知识卡片格式规范

每条知识卡片必须包含：

```markdown
---
title: 标题（临床问题/主诉形式）
tags: [标签1, 标签2]
level: [指南]|[教科书]|[临床经验]|[综述]
author: 作者名
updated: YYYY-MM-DD
related: [相关卡片标题1, 相关卡片标题2]
---

# 标题

## 结论
1-3句话直接回答核心临床问题

## 核心要点
- 3-5条精炼要点
- 可快速扫读

## 来源
- 指南/教科书名称（年份）
- 链接（如有）

## 相关
- [链接到其他卡片](path)
```

### 格式要求

- **结论先行**：打开页面3秒内看到答案
- **极短卡**（急症/药物）：300-500字，不超过一屏
- **标准卡**（疾病/检验）：500-1500字
- **必须引用来源**：每条至少一个可查来源
- **不写机制长篇**：这里不是教科书
- **不写患者信息**：保护隐私

## 提交规范

- 新增内容：`feat: add [分类] 标题`
- 修正内容：`fix: 标题 - 修正说明`
- 格式调整：`style: 说明`

## 许可

提交的贡献默认按 CC BY-NC-SA 4.0 许可发布。
