## 这是什么

**规培笔记** 是一个面向中国年轻临床医师的开放知识库。每一篇都以"知识卡片"形式呈现——打开页面3秒内看到结论，5条要点快速扫读，每条信息标注来源和证据等级。

> 不是教科书替代品。是你在值班凌晨3点遇到问题、打开手机就能搜到的实用答案。

## 目标用户

| 用户 | 场景 |
|------|------|
| 🩺 **规培医师（核心）** | 独立值班时的速查参考——低血压了怎么办？这个化验单怎么判读？ |
| 📋 **专硕研究生** | 临床轮转中快速定位知识锚点，同时兼顾科研和考试 |
| 🎓 **实习医师** | 从教科书到临床的第一座桥——学了理论，但床边怎么做？ |

## 知识卡片格式

每条知识严格遵循"结论先行"原则：

```markdown
# 急性低血压的评估与处理          ← 临床问题

## 结论                             ← 打开3秒内看到核心答案
收缩压<90mmHg或MAP<65mmHg持续>5分钟
→ 立即评估组织灌注 → 按休克分型处理。

## 核心要点                         ← 3-5条精炼要点，快速扫读
- 第一步：确认血压真实（复测+对侧上肢）
- 第二步：快速评估灌注（意识、尿量、皮肤、乳酸）
- 补液试验：晶体液500mL iv 15min
- 去甲肾上腺素：起始0.05-0.1 μg/kg/min

## 来源                             ← 每条必有来源
- Surviving Sepsis Campaign 2024 指南
- 中国脓毒症急诊治疗指南（2024）

## 相关                             ← 站内知识网络
- [休克分型](../emergency/shock-types.md)
```

### 核心原则

| 原则 | 说明 |
|------|------|
| 📌 **结论先行** | 打开页面3秒内看到答案，后面才是推导 |
| 📌 **来源可溯** | 每条标注指南/教科书名称+年份。无来源不发布 |
| 📌 **证据可视** | `[指南]` `[教科书]` `[临床经验]` `[综述]` 四种简标 |
| 📌 **场景驱动** | 文件按临床场景组织，不从解剖学目录出发 |

### 内容边界

- ✅ **写**：临床场景下的实用知识——怎么判断、怎么处理、用什么药、注意什么
- ❌ **不写**：教科书已有的长篇机制讲解、超说明书用药（除非明确标注+引用依据）、任何具体患者信息

## 如何使用

### 🌐 在线访问

部署至 Cloudflare Pages 后可通过域名直接访问。支持：
- 🔍 **全文搜索**（右上角框或 Ctrl+K）
- 🌙 **深色模式**（夜间值班友好）
- 📱 **移动端适配**（手机浏览器直接查）
- 💬 **评论区勘误**（Giscus · 需GitHub登录）

### 🖥️ 本地运行

```bash
git clone https://github.com/ayaosmile/residency-notes.git
cd residency-notes
npm install
npm run docs:dev     # 启动开发服务器 → http://localhost:5173
```
